# 全球气候形成的综合知识框架

---

## 总论：五大驱动因素的层级关系

全球气候是**五大核心因素**共同作用的产物。它们并非平等并列，而是存在清晰的层级结构：

| 层级 | 因素 | 角色 | 性质 |
|------|------|------|------|
| **第一层** | 纬度（太阳辐射） | 根本能量来源，决定热量基础 | 地带性（沿纬线呈带状） |
| **第二层** | 大气环流 | 热量与水分的再分配机制 | 地带性为主 |
| **第二层** | 海陆位置 | 水汽来源与热力差异 | 非地带性/半地带性 |
| **第三层** | 洋流 | 对沿岸气候的叠加修正 | 半地带性 |
| **修正层** | 地形 | 打乱所有地带性规律的"搅局者" | 非地带性 |

**核心逻辑链**：
> 纬度决定"有多少太阳能量进入该系统" → 大气环流决定"能量和水分如何在全球尺度上重新分配" → 海陆位置决定"水汽能否到达、大陆性有多强" → 洋流对沿岸地区进行"热量调节" → 地形在局地尺度上"推翻一切规则"

---

## 一、纬度：气候系统的"总开关"

### 1.1 物理机制

太阳辐射是地球气候系统的**唯一外部能量来源**。由于地球的球体形状，不同纬度接受的太阳辐射强度差异巨大：

- **低纬度（0°-30°）**：太阳高度角大，辐射集中，年辐射收入 > 支出（能量盈余区）
- **中纬度（30°-60°）**：辐射收支大致平衡（过渡区）
- **高纬度（60°-90°）**：太阳高度角小，辐射分散，年辐射收入 < 支出（能量亏损区）

这种辐射不平衡是**一切大气运动与洋流的根本驱动力**——大气环流和洋流本质上就是地球将低纬度盈余热量输送到高纬度的"运输系统"。

### 1.2 由纬度决定的热量带

| 温度带 | 纬度范围 | 年均温 | 气候类型举例 |
|--------|----------|--------|-------------|
| 热带 | 0°-23.5° | >20°C | 热带雨林、热带草原、热带沙漠、热带季风 |
| 亚热带 | 23.5°-35° | 15-20°C | 亚热带季风、地中海气候 |
| 温带 | 35°-55° | 0-15°C | 温带季风、温带海洋性、温带大陆性 |
| 亚寒带 | 55°-66.5° | <0°C（年均） | 亚寒带针叶林气候 |
| 寒带 | 66.5°-90° | <0°C（最热月<10°C） | 苔原、冰原气候 |

### 1.3 纬度因素的局限性

纬度只能解释**为什么赤道热、极地冷**，但无法解释：
- 为什么副热带地区有大片沙漠（撒哈拉、阿拉伯、澳大利亚）？
- 为什么同纬度的伦敦（51°N）和莫斯科（55°N）冬季温差可达15°C以上？
- 为什么赤道附近的东非高原不是热带雨林而是热带草原？

这些问题需要后续因素来回答。

## 二、大气环流：热量与水分的"全球调度系统"

大气环流是理解气候分布的**最关键因素**。它决定了：
1. **哪里盛行上升气流（多雨）**，哪里盛行下沉气流（干燥）
2. **风从哪个方向吹来**（从海洋带来水汽 or 从内陆带来干燥空气）
3. **季风环流**如何在特定大陆位置形成季节性的干湿交替

### 2.1 三圈环流与气压带

地球自转产生的科里奥利力，将赤道-极地的单一热力环流"撕裂"为三个环流圈：

```
极地高压 ───────────────────────────── 90°N
     ↓ 极地东风带
副极地低压 ──────────────────────────── 60°N  ← 极锋：冷暖空气交汇，多气旋雨
     ↓ 盛行西风带
副热带高压 ──────────────────────────── 30°N  ← 下沉气流，干燥少雨
     ↓ 东北信风带
赤道低压（ITCZ）──────────────────────── 0°   ← 强烈上升，终年多雨
     ↓ 东南信风带
副热带高压 ──────────────────────────── 30°S
     ↓ 盛行西风带
副极地低压 ──────────────────────────── 60°S
     ↓ 极地东风带
极地高压 ───────────────────────────── 90°S
```

### 2.2 气压带与降水的对应关系

| 气压带 | 成因 | 气流方向 | 降水特征 | 对应的典型气候 |
|--------|------|----------|----------|---------------|
| **赤道低压带** (ITCZ) | 热力：强烈受热上升 | ↑ 上升 | 终年多雨 | 热带雨林气候 |
| **副热带高压带** (~30°) | 动力：高空堆积下沉 | ↓ 下沉 | 终年干燥 | 热带沙漠气候、地中海气候（夏） |
| **副极地低压带** (~60°) | 动力：冷暖交汇抬升 | ↑ 上升 | 多气旋雨 | 亚寒带气候（降水偏多） |
| **极地高压** (~90°) | 热力：寒冷收缩下沉 | ↓ 下沉 | 终年干燥 | 冰原气候 |

### 2.3 风带与大陆东西岸的"命运分歧"

**这是理解气候分布格局最核心的概念之一**。

#### 低纬度（0°-30°）：信风带

- **大陆东岸**：信风从海洋吹向陆地 → 带来水汽 → **湿润**（如巴西东南部、马达加斯加东部、澳大利亚东北部）
- **大陆西岸**：信风从陆地吹向海洋（离岸风）→ 干燥 + 副高控制 → **极端干旱**（如撒哈拉西海岸、纳米布沙漠、秘鲁-智利北部）

#### 中纬度（30°-60°）：盛行西风带

- **大陆西岸**：西风从海洋吹向陆地 → 终年湿润 → **温带海洋性气候**（如西欧、北美西北海岸、新西兰、智利南部）
- **大陆东岸**：西风从陆地吹向海洋（离岸风）→ 降水偏少 → 多形成**温带大陆性气候**的内陆部分

#### 为什么西欧比同纬度的东亚温暖得多？

- **西欧（巴黎 48°N）**：位于大陆西岸，终年受盛行西风控制，西风从大西洋带来暖湿空气，加之北大西洋暖流的影响 → 冬季温和（1月均温约4°C）
- **东亚（哈尔滨 45°N）**：位于大陆东岸，冬季受冬季风（西北季风）控制，风从西伯利亚内陆吹来 → 冬季严寒（1月均温约-19°C）

### 2.4 季风环流：打破三圈环流的"季节性反转"

季风是大尺度海陆热力差异驱动的**季节性风向反转**。它是理解东亚、南亚、东南亚、西非气候的关键。

#### 季风的形成机制

**冬季**：大陆冷却快 → 形成冷高压（如西伯利亚高压）→ 风从陆地吹向海洋 → 干燥少雨

**夏季**：大陆加热快 → 形成热低压（如亚洲低压）→ 风从海洋吹向陆地 → 带来大量降水

#### 全球主要季风系统

| 季风区 | 夏季风向 | 水汽来源 | 影响地区 |
|--------|----------|----------|----------|
| **东亚季风** | 东南风 | 太平洋 | 中国东部、朝鲜半岛、日本 |
| **南亚季风** | 西南风（东南信风越过赤道偏转） | 印度洋 | 印度、孟加拉国、缅甸、中南半岛 |
| **西非季风** | 西南风 | 大西洋（几内亚湾） | 西非萨赫勒以南 |
| **澳大利亚北部季风** | 西北风（北半球东北信风越过赤道偏转） | 印度洋/太平洋 | 澳大利亚北部 |

#### 季风区与非季风区的根本区别

- **非季风区**（如西欧）：全年受同一种风带控制，降水季节分配较均匀
- **季风区**（如东亚）：风向随季节反转 → 降水高度集中于夏季 → "雨热同期"是东亚季风气候最突出的特点

#### 为什么南亚季风（西南季风）比东亚季风更强烈？

南亚季风的夏季风为西南风，其根本驱动力不仅来自海陆热力差异，还叠加了**行星风带的季节性移动**：

1. 北半球夏季，ITCZ北移至~20°N-25°N
2. 南半球的东南信风越过赤道后，受科里奥利力（北半球向右偏）的作用，偏转为西南风
3. 这支"越赤道气流"携带印度洋大量水汽冲向青藏高原南麓
4. 青藏高原的**热力作用**（夏季高原面强烈加热）加强了上升运动
5. 喜马拉雅山脉的**地形阻挡**迫使气流抬升 → 形成世界上最大的降水带之一（乞拉朋齐年降水量可达11000mm以上）

### 2.5 大气环流在不同"大陆位置"的主导性总结

| 大陆位置 | 主导环流 | 气候特征 | 典型地区 |
|----------|----------|----------|----------|
| **大陆西岸（低纬）** | 副热带高压 + 离岸信风 | 干旱，多沙漠 | 撒哈拉西岸、秘鲁沿岸、纳米布 |
| **大陆西岸（中纬）** | 盛行西风 | 终年温和湿润 | 西欧、北美西北海岸、新西兰 |
| **大陆西岸（中低纬过渡）** | 副高（夏）+ 西风（冬）| 夏干冬湿：地中海气候 | 地中海沿岸、加州、智利中部、南非开普 |
| **大陆东岸（低纬）** | 信风（向岸风） | 温暖湿润 | 巴西东南沿海、马达加斯加东部 |
| **大陆东岸（中低纬）** | 季风环流 | 夏季多雨、冬季干燥 | 东亚、南亚、东南亚 |
| **大陆东岸（中高纬）** | 季风（冬）或内陆风 | 冬季寒冷干燥 | 中国东北、俄罗斯远东 |
| **大陆内部** | 远离海洋，任何风向都已失去水汽 | 干燥，年/日温差大 | 中亚、蒙古、美国大平原西部 |

## 三、海陆位置：水汽的"可及性"与大陆度

### 3.1 海陆热力性质差异

这是驱动季风的根本原因，也决定了气候的"海洋性"与"大陆性"程度：

| 物理性质 | 海洋 | 陆地 |
|----------|------|------|
| 比热容 | 大（约4.2 J/g·°C） | 小（约0.8-1.5 J/g·°C） |
| 透光性 | 太阳辐射可穿透至数十米深 | 辐射仅加热地表薄层 |
| 蒸发散热 | 大量热量用于蒸发 | 蒸发量有限 |
| 热量传输 | 洋流和湍流混合将热量向下输送 | 热量主要靠传导，向下传输极慢 |

**结果**：海洋升温慢、降温也慢（"热惯性"大）；陆地升温快、降温也快。这导致：
- **海洋性气候**：年温差小，冬暖夏凉，降水较均匀（如伦敦：年温差约13°C）
- **大陆性气候**：年温差大，冬冷夏热，降水集中于夏季（如莫斯科：年温差约29°C；西伯利亚内陆年温差可达60-100°C）

### 3.2 距海远近与降水

一般规律：距海越远，降水量越少，气候越趋向干旱。

**具体例子——欧亚大陆从西到东的降水变化**：

| 位置 | 城市 | 年降水量 | 主导因素 |
|------|------|----------|----------|
| 大陆西岸 | 伦敦 | ~600mm | 西风从大西洋带来水汽 |
| 中西欧内陆 | 华沙 | ~550mm | 距海渐远，但西风仍有影响 |
| 东欧内陆 | 莫斯科 | ~700mm | 仍有大西洋水汽到达（西风无地形阻挡） |
| 西西伯利亚 | 鄂木斯克 | ~350mm | 距海遥远，水汽耗尽 |
| 东西伯利亚 | 雅库茨克 | ~240mm | 极端大陆性，加之周围山脉阻挡 |
| 大陆东岸 | 符拉迪沃斯托克（海参崴） | ~800mm | 夏季东南季风带来太平洋水汽 |

### 3.3 海陆位置的关键变量不是"距海绝对距离"，而是"相对盛行风向的位置"

- **西欧**：盛行西风+位于大陆西岸=终年海洋水汽补给=温带海洋性气候
- **撒哈拉西岸**：东北信风（离岸风）+位于大陆西岸=极端干旱
- **澳大利亚东岸**：东南信风（向岸风）+位于大陆东岸=湿润
- **澳大利亚西岸**：东南信风（离岸风）+位于大陆西岸=干旱沙漠

## 四、洋流：沿岸气候的"热量调节器"

### 4.1 洋流的成因类型

| 类型 | 驱动力 | 特征 | 举例 |
|------|--------|------|------|
| **风海流** | 盛行风的摩擦力 | 表层洋流主体 | 北赤道暖流、西风漂流 |
| **补偿流** | 海水连续性（水平/垂直） | 上升流或下降流 | 秘鲁上升流、加利福尼亚上升流 |
| **密度流** | 温盐差异引起的密度梯度 | 深层洋流 | 北大西洋深层水（NADW）形成 |

### 4.2 暖流与寒流的气候效应

#### 暖流：增温增湿

| 暖流名称 | 流经区域 | 气候影响 |
|----------|----------|----------|
| **北大西洋暖流（墨西哥湾暖流的延伸）** | 西欧沿岸，最北可达巴伦支海 | 使西欧冬季温度比同纬度高10-15°C，挪威沿岸港口终年不冻 |
| **黑潮（日本暖流）** | 日本太平洋沿岸 | 使日本太平洋侧冬季较温暖，降水偏多 |
| **巴西暖流** | 巴西南部-阿根廷北部 | 增加沿岸降水 |
| **莫桑比克暖流** | 东南非洲沿岸 | 使马达加斯加东部和莫桑比克降水充沛 |

#### 北大西洋暖流对西欧气候的决定性影响（详细分析）

这是全球最著名的洋流-气候案例：

- **不列颠群岛（50°-60°N）**：1月均温约4-6°C，而同位纬度的拉布拉多半岛（加拿大东北部）1月均温约-20至-25°C
- **挪威沿岸（~70°N）**：1月均温仍在0°C左右，港口全年不冻；而同纬度的西伯利亚北冰洋沿岸为永久冻土
- **摩尔曼斯克（69°N）**：世界最北的不冻港，而同位纬度的格陵兰东岸终年冰封

**机制**：墨西哥湾暖流每年向北输送约3×10¹⁶W的热量（相当于全球人类能源消耗的数百倍），这些热量通过盛行西风传递到西欧大气中，使得西欧成为同纬度最温暖的地区。

#### 寒流：降温减湿

| 寒流名称 | 流经区域 | 气候影响 |
|----------|----------|----------|
| **秘鲁寒流（洪堡寒流）** | 秘鲁-智利北部沿岸 | 使南美洲西岸5°-30°S极度干旱，形成阿塔卡马沙漠 |
| **本格拉寒流** | 纳米比亚-安哥拉沿岸 | 使纳米布沙漠延伸至海岸线 |
| **加利福尼亚寒流** | 美国加利福尼亚-下加利福尼亚 | 使加州南部沿海较为干燥 |
| **加那利寒流** | 摩洛哥-毛里塔尼亚沿岸 | 加剧撒哈拉沙漠的西延 |
| **西澳大利亚寒流** | 澳大利亚西岸 | 澳大利亚西部沙漠的成因之一 |

#### 寒流+副热带高压=世界上最干旱的海岸线

副热带大陆西岸几乎是全球沙漠的"固定配方"：

1. **副热带高压**：终年盛行下沉气流 → 空气绝热增温 → 相对湿度降低 → 难以成云致雨
2. **信风为离岸风**：风从陆地吹向海洋 → 无法带来海洋水汽
3. **寒流经过**：近海面空气冷却 → 形成逆温层 → 对流受到抑制 → 降水进一步减少

**南美洲西岸的阿塔卡马沙漠**是这一配方的极致体现：
- 副高控制
- 东南信风为离岸风
- 秘鲁寒流强烈降温
- 安第斯山脉阻挡了来自大西洋的湿气
- 结果：部分气象站从未记录到降水，有些地区年降水量<1mm

### 4.3 上升流与沿岸干旱

寒流区通常伴随**上升流**（离岸风将表层海水吹离海岸，深层冷水上涌补充）。上升流区海水温度低，大气底层冷却形成逆温，进一步抑制对流和降水。

但上升流有一个重要的**生态效应**：深层水富含营养盐 → 浮游生物繁盛 → 世界著名渔场（秘鲁渔场、本格拉渔场、加利福尼亚渔场）。

## 五、地形：非地带性的最终"修正符"

地形是打破所有纬度-环流-洋流规律的**最强修正因素**。一条山脉可以彻底改变两侧的气候，在数十公里的距离内创造出完全不同的气候类型。

### 5.1 地形对气候的五大修正机制

| 机制 | 原理 | 典型效应 |
|------|------|----------|
| **地形抬升（地形雨）** | 湿空气遇山抬升→绝热冷却→水汽凝结→迎风坡多雨 | 喜马拉雅南麓、安第斯山西坡 |
| **雨影效应（焚风）** | 空气越山下沉→绝热增温→相对湿度骤降→背风坡干燥 | 安第斯山东侧的巴塔哥尼亚、落基山东侧的大平原 |
| **海拔降温（气温直减率）** | 海拔每升1000m，气温降约6.5°C | 赤道高山出现永久积雪（乞力马扎罗山） |
| **屏障效应** | 山脉阻挡气团移动 | 秦岭阻挡北方冷空气南下，使四川盆地冬季温暖 |
| **热力效应（高原加热/冷却）** | 高原夏季为热源（加强上升），冬季为冷源（加强高压） | 青藏高原对亚洲季风的加强作用 |

### 5.2 经典案例

#### 案例1：安第斯山脉对南美洲气候的重塑

安第斯山脉纵贯南美洲西侧，海拔普遍在4000m以上，对南美洲气候产生了全方位的修正：

| 位置 | 安第斯山脉的影响 |
|------|-----------------|
| **西坡（智利南部 40°-55°S）** | 终年迎盛行西风 → 地形抬升 → 年降水量3000-5000mm → 温带海洋性气候（温带雨林） |
| **东坡（阿根廷巴塔哥尼亚 40°-55°S）** | 位于西风背风坡 → 空气下沉增温 → 年降水量<300mm → 温带荒漠/半荒漠（雨影沙漠） |
| **西坡（秘鲁-智利北部 5°-30°S）** | 山脉与海岸平行 → 阻挡大西洋水汽 + 副高+寒流 → 阿塔卡马沙漠 |
| **东坡（玻利维亚-阿根廷北部）** | 东南信风从大西洋带来水汽遇到安第斯山东坡抬升 → 形成亚马孙河上游的丰富降水 |

**东西坡对比**：智利南部的蒙特港（西坡，41°S）年降水量约2200mm；而仅隔一座安第斯山、阿根廷的圣卡洛斯-德巴里洛切（东坡，41°S）年降水量仅约500mm。

#### 案例2：青藏高原——地球上最强大的地形-气候互作系统

青藏高原（平均海拔>4000m，面积约250万km²）对全球气候的影响远超其所在区域：

**（a）动力阻挡作用**
- 将西风带分裂为南北两支，北支绕过高原北侧，南支绕过高原南侧
- 冬季阻挡西伯利亚冷空气南下，使南亚比同纬度其他地区温暖
- 使东亚冬季风不能到达南亚，南亚气候受印度洋和西风南支控制

**（b）热力作用（高原季风）**
- 夏季：高原面强烈加热 → 形成热低压 → 加强了来自印度洋的西南季风
- 冬季：高原面冷却 → 形成冷高压 → 加强了高原南侧的冬季下沉气流
- 高原热力作用是南亚季风如此强烈的重要原因之一

**（c）水汽通道作用**
- 西南季风经孟加拉湾向东北输送 → 被喜马拉雅山南坡抬升 → 形成世界上降水量最大的地区之一（印度梅加拉亚邦的乞拉朋齐和毛辛拉姆）

#### 案例3：北美洲科迪勒拉山系的三重气候分异

| 区域 | 气候 | 机制 |
|------|------|------|
| **海岸山脉西坡（40°-60°N）** | 温带海洋性气候（温带雨林） | 终年迎西风 → 降水量2000-4000mm |
| **山间高原盆地（大盆地）** | 温带荒漠/半荒漠 | 喀斯喀特山脉和内华达山脉的双重雨影 |
| **落基山东侧大平原** | 温带大陆性半干旱 | 西风经数道山脉阻挡 → 水汽耗尽 |

北美洲西岸与西欧同纬度，但温带海洋性气候的范围远小于西欧。原因在于**北美山脉呈南北走向**，西风带来的水汽在海岸山脉西坡即被截留，无法深入内陆；而**欧洲山脉以东西走向为主**（阿尔卑斯山、喀尔巴阡山），不阻挡西风东进，使得海洋性影响可以深入内陆上千公里。

#### 案例4：秦岭——一条山脉界定的气候分界线

秦岭（平均海拔2000-3000m，东西走向）是中国最重要的气候分界线之一：

| 对比项 | 秦岭以北（西安） | 秦岭以南（汉中） |
|--------|-----------------|-----------------|
| 1月均温 | 约0°C | 约3°C |
| 年降水量 | ~550mm | ~850mm |
| 气候类型 | 温带季风/半湿润 | 亚热带季风/湿润 |
| 典型植被 | 落叶阔叶林 | 常绿阔叶林 |
| ≥10°C积温 | <4500°C | >4500°C |

秦岭阻挡了冬季冷空气南下，也阻挡了夏季暖湿气流北上，使得山脉两侧仅约100km的距离就产生了从"温带"到"亚热带"的气候跃变。

## 六、五大因素的协同作用：综合案例分析

### 6.1 撒哈拉沙漠——为什么位于这个纬度、这个位置？

| 因素 | 作用 |
|------|------|
| **纬度** | 位于15°-30°N，太阳辐射强烈，蒸发旺盛 |
| **大气环流** | 终年受副热带高压控制，盛行下沉气流；东北信风为离岸风（从陆地吹向大西洋） |
| **海陆位置** | 北非大陆东西宽阔，西部距大西洋虽近但信风为离岸风，东部距海遥远 |
| **洋流** | 西岸为加那利寒流（降温减湿，形成逆温层抑制降水） |
| **地形** | 地势整体平坦，无大型山脉产生抬升降水 |

→ **综合结果**：纬度提供热力基础；大气环流是主导因素（副高+离岸信风）；寒流叠加干燥效应；海陆位置不利水汽输入；地形不制造降水机会 = **地球上最大的连续沙漠**

### 6.2 亚马孙雨林——为什么是地球上最大的热带雨林？

| 因素 | 作用 |
|------|------|
| **纬度** | 大部分位于赤道附近（5°N-10°S），太阳辐射终年充足 |
| **大气环流** | 赤道低压带（ITCZ）控制，终年盛行上升气流 → 对流雨旺盛 |
| **海陆位置** | 北临大西洋，东临大西洋，东南信风和东北信风均从大洋带来大量水汽 |
| **洋流** | 南赤道暖流和圭亚那暖流流经 → 增加近海水汽蒸发 |
| **地形** | **关键因素**：安第斯山脉位于大陆西缘，从大西洋来的湿气一路向西深入内陆，遇安第斯山东坡抬升 → 形成大量地形雨 → 使雨林向西延伸至大陆深处（直到安第斯山脚） |

→ **综合结果**：如果没有安第斯山脉的地形抬升，亚马孙雨林可能只局限在南美洲东部沿海，不会成为横贯大陆的"地球之肺"。

### 6.3 东亚季风气候——大气环流+海陆位置+地形的经典协奏

| 因素 | 作用 |
|------|------|
| **纬度** | 覆盖热带至寒温带（18°-53°N），形成从热带季风到寒温带季风的完整系列 |
| **大气环流** | 冬季受西伯利亚高压控制（冬季风：西北风，干冷）；夏季受亚洲低压和西太平洋副高引导（夏季风：东南风，暖湿） |
| **海陆位置** | 位于世界最大大陆（亚欧大陆）和最大大洋（太平洋）的交界处 → 海陆热力差异最大化 → 季风强度最大 |
| **洋流** | 黑潮（暖流）流经东海岸 → 增加夏季水汽供应；冬季近岸有沿岸寒流（中国沿岸流） |
| **地形** | 青藏高原的存在强化了亚洲低压和季风环流；秦岭-大别山等东西向山脉阻挡冬季风南下，形成气候分界 |

→ **综合结果**：东亚季风区拥有全球同纬度中**最显著的四季分明**、**雨热同期**（夏季高温与多雨同时出现，对农业生产极为有利），以及**从南到北最完整的气候带谱**。冬季风使同纬度东亚比西欧冷得多；夏季风又使东亚夏季比西欧热得多且降水更集中。

### 6.4 伦敦 vs 莫斯科（同纬度、不同命运）

| 因素 | 伦敦（51°N） | 莫斯科（55°N） |
|------|-------------|----------------|
| **纬度** | 相近，太阳辐射总量相似 | 相近 |
| **大气环流** | 终年受盛行西风控制 | 受西风影响但较弱，冬季受大陆高压影响 |
| **海陆位置** | 大陆西岸，紧邻大西洋 | 大陆内部，距大西洋>1500km |
| **洋流** | 北大西洋暖流直接流经 | 无洋流直接影响 |
| **地形** | 无山脉阻隔西风 | 地形平坦，但距离太远水汽已耗尽 |

→ **结论**：即便纬度相同，海陆位置+洋流+盛行风向的组合可以产生完全不同的气候。

## 七、方法论总结：分析一个地区气候的系统步骤

当你面对任何一个地区的气候分析时，按照以下顺序逐层叠加，而非"一次性全部考虑"：

```
步骤1【纬度】：确定热量带基础（热带/亚热带/温带/寒带）

步骤2【大气环流】：确定该地常年受哪个气压带/风带控制，是否有季风反转
   ├── 常年受单一风带控制？ → 气候年变化较平稳
   ├── 随气压带季节性移动而交替？ → 形成干湿季或夏干冬湿（地中海式）
   └── 有季风反转？ → 形成冬夏显著差异的季风气候

步骤3【海陆位置+步骤2的盛行风向】：
   ├── 位于盛行风向的向风岸？ → 水汽充足
   ├── 位于盛行风向的背风岸？ → 干燥
   ├── 距海遥远？ → 大陆性增强
   └── 近海但有离岸风？ → 海洋水汽无法到达

步骤4【洋流】：沿岸是否有暖流/寒流经过？
   ├── 暖流 → 增温增湿
   └── 寒流 → 降温减湿，叠加干旱

步骤5【地形】：局地修正
   ├── 是否有山脉阻挡？ → 迎风坡多雨/背风坡雨影
   ├── 海拔高度？ → 温度直减率修正
   ├── 山脉走向与盛行风向的关系？
   └── 高原热力/动力效应？
```

**最重要的原则**：五大因素中，第1层（纬度）决定了可能性范围，第2-3层（环流+海陆）决定了基本气候类型，第4层（洋流）在特定区域叠加强化或抑制效应，第5层（地形）可以在任何环节推翻前面的所有预期。
