# 柯本（Köppen）气候类型判定矩阵

## 基于五大驱动因素的气候类型成因解构

---

## 前言：如何使用本矩阵

本矩阵将柯本气候分类法的每一主要类型，按照五大驱动因素进行**系统化解构**。每个气候类型包含：

- **柯本符号与判定标准**（温度、降水阈值）
- **五大因素的典型组合**（该类型最常见的成因配方）
- **地理分布规律**（该类型在全球的典型位置）
- **成因机制链**（从根本驱动力到最终气候表现的因果链）

阅读时请注意：同一气候类型在不同地区可能由不同的因素组合形成（例如，BWh可以出现在副热带西岸，也可以出现在温带内陆雨影区）。本矩阵给出的是**最常见的典型组合**。

---

## 一、柯本气候分类法速查

### 1.1 基本符号系统

```
第一字母（主类）  第二字母（降水型）     第三字母（温度型）
A = 热带         f = 终年湿润            a = 夏季炎热
B = 干燥         w = 冬季干燥            b = 夏季温暖
C = 暖温带       s = 夏季干燥            c = 夏季凉爽
D = 冷温带       m = 季风（仅用于A类）    d = 冬季严寒
E = 极地         W = 沙漠（仅用于B类）    
                 S = 草原（仅用于B类）    
                 T = 苔原（仅用于E类）    
                 F = 冰原（仅用于E类）    
```

### 1.2 柯本判定核心阈值速查

| 参数 | 阈值 | 含义 |
|------|------|------|
| 热带判定 | 最冷月均温 ≥ 18°C | 无真正的"冬季" |
| 干燥判定 | 年降水量 < 潜在蒸散量 | B类气候的核心条件 |
| 暖温带 vs 冷温带 | 最冷月均温：0°C ≤ T < 18°C（C类）；T < 0°C（D类） | 有无月均温低于冰点 |
| 夏季炎热( a ) | 最热月均温 ≥ 22°C | C/D类温度细分 |
| 夏季温暖( b ) | 最热月均温 < 22°C，但有 ≥4个月均温≥10°C | C/D类温度细分 |
| 夏季凉爽( c ) | ≥4个月均温≥10°C，但最热月<22°C | C/D类温度细分（高纬度/高海拔） |
| 冬季严寒( d ) | 最冷月均温 < -38°C | 仅D类，极端大陆性 |

**干燥气候（B类）的精确判定公式**：
- 若降水集中于夏季：`R < 2 × T + 28`（R=年降水量cm, T=年均温°C）
- 若降水集中于冬季：`R < 2 × T`
- 若降水均匀分布：`R < 2 × T + 14`
- 其中 `R < 上述阈值` 且 `R ≥ 阈值/2` → **BS（草原）**；`R < 阈值/2` → **BW（沙漠）**

## 二、A类 热带气候（最冷月均温 ≥ 18°C）

### 2.1 Af — 热带雨林气候（Tropical Rainforest）

| 维度 | 典型特征 |
|------|----------|
| **柯本标准** | 最冷月均温 ≥ 18°C；最干月降水量 ≥ 60mm（无真正的干季） |
| **全球占比** | 约陆地面积的9% |

#### 五大驱动因素

| 因素 | 典型组合 | 成因说明 |
|------|----------|----------|
| **纬度** | 0° - 10° N/S | 赤道附近，太阳辐射终年充足，年均温25-28°C，年温差极小（<3°C） |
| **大气环流** | 终年受**赤道低压带（ITCZ）**控制 | 盛行强烈上升气流 → 对流雨旺盛 → 午后雷阵雨几乎每天发生；ITCZ年摆动范围主要覆盖赤道附近，因此该区域终年在ITCZ影响范围内 |
| **海陆位置** | 常位于**沿海低地或大型岛屿**（亚马孙、刚果、马来群岛） | 海洋提供持续的水汽补给；热带海洋的蒸发量极大 → 空气始终接近饱和 |
| **洋流** | 沿岸常有**暖流经过** | 巴西暖流（亚马孙河口）、几内亚湾暖流（刚果盆地西侧）、赤道逆流（印尼群岛）→ 暖流增强蒸发 → 提供更充足的水汽 |
| **地形** | **迎风坡**增强降水 | 信风从海洋吹向陆地，遇山脉/高地抬升 → 地形雨叠加对流雨 → 年降水量可达3000-10000mm（如夏威夷怀厄莱阿莱峰年降水11684mm，是全球降水极值之一） |

#### 典型地区
| 地区 | 具体位置 | 关键成因 |
|------|----------|----------|
| 亚马孙盆地 | 南美洲5°N-10°S | ITCZ + 东南信风从大西洋带水汽 + 安第斯山东坡抬升 |
| 刚果盆地 | 非洲中部赤道两侧 | ITCZ + 大西洋暖湿气流 |
| 马来群岛/印尼 | 东南亚赤道海域 | 海洋性 + ITCZ + 赤道逆流暖水 + 各岛屿地形抬升 |
| 马达加斯加东部 | 非洲东南印度洋 | 东南信风（向岸风）+ 山地迎风坡 + 暖流（莫桑比克暖流） |

#### 成因机制链
```
赤道太阳辐射强 → 终年高温 → ITCZ终年控制 → 上升气流 + 对流旺盛
    ↓
暖流经过 + 海洋水汽充足 → 空气湿度极高
    ↓
（如有地形抬升）→ 地形雨叠加 → 年降水 2000-10000+mm
    ↓
结果：Af 热带雨林气候 → 终年高温多雨，无明显干季
```

#### Af 的特例：偏离赤道的热带雨林

马达加斯加东部（12°-25°S）不在赤道附近，却发育了Af气候，原因是：
- **强向岸信风**（东南信风从印度洋带来水汽）补偿了ITCZ的缺失
- **地形抬升**（马达加斯加中央高地东坡）产生持续地形雨
- **暖流**（莫桑比克暖流/南赤道暖流）增强蒸发
- 这是**环流+地形+洋流"合谋"取代了纬度的主导性**的典型例子

### 2.2 Am — 热带季风气候（Tropical Monsoon）

| 维度 | 典型特征 |
|------|----------|
| **柯本标准** | 最冷月均温 ≥ 18°C；有短暂的干季（最干月降水量 < 60mm），但年降水量足够大，干季不足以形成真正的干旱 |

#### 五大驱动因素

| 因素 | 典型组合 | 成因说明 |
|------|----------|----------|
| **纬度** | 5° - 15° N（主要在北半球） | 仍在热带范围内，但离赤道较远，ITCZ仅夏季到达 |
| **大气环流** | **季风环流** + ITCZ季节移动 | **夏季**：ITCZ北移 + 海陆热力差异 → 越赤道气流转为西南/东南季风 → 暴雨；**冬季**：ITCZ南移 → 受信风或大陆气流控制 → 短暂干季 |
| **海陆位置** | 大陆**东岸/南岸**，面对暖洋 | 如印度西岸（面对阿拉伯海）、缅甸-泰国西岸（面对孟加拉湾）、西非几内亚湾沿岸 |
| **洋流** | 夏季风路径上有**暖流** | 暖海水提供大量蒸发 → 季风携带充沛水汽登陆 |
| **地形** | **迎风坡/山脉屏障**是关键放大因素 | 喜马拉雅山南麓（印度乞拉朋齐）、缅甸若开山脉西坡——湿季风被山脉强制抬升 → 产生地球上最强的季风雨 |

#### 典型地区
| 地区 | 具体位置 | 关键成因 |
|------|----------|----------|
| 印度西海岸（马拉巴尔海岸） | 印度半岛西岸 | 西南季风+西高止山脉迎风坡 |
| 孟加拉国-缅甸沿岸 | 孟加拉湾东北岸 | 西南季风+若开山脉抬升 |
| 西非沿岸（几内亚-塞拉利昂） | 非洲西岸5°-10°N | 西非季风（西南风从几内亚湾带来水汽） |

#### 成因机制链
```
热带纬度 + 大陆-海洋巨大热力对比 → 夏季季风强烈爆发
    ↓
夏季风从暖洋面带来巨量水汽
    ↓
（关键）山脉迎风坡强烈抬升 → 极端暴雨（最湿月降水可达 2000-3000mm）
    ↓
冬季：ITCZ南移 + 离岸风 → 降水锐减 → 短暂干季（2-3个月）
    ↓
结果：Am 热带季风气候 → 降水极集中于夏季，短干季
```

### 2.3 Aw/As — 热带干湿季气候 / 热带草原气候（Tropical Savanna）

| 维度 | 典型特征 |
|------|----------|
| **柯本标准** | 最冷月均温 ≥ 18°C；有明显的干季（冬季），最干月降水量 < 60mm，且干季长度足以使年降水总量不足以维持雨林 |

#### 五大驱动因素

| 因素 | 典型组合 | 成因说明 |
|------|----------|----------|
| **纬度** | 8° - 20° N/S（热带向副热带过渡带） | 处于ITCZ夏季影响范围的边缘 |
| **大气环流** | **ITCZ与副热带高压的交替控制** | **夏季（高太阳期）**：ITCZ到达 → 对流雨 + 季风/信风降水；**冬季（低太阳期）**：ITCZ远离 → 副热带高压控制 → 下沉气流 → 明显干季。干季长度随纬度增加而延长（离赤道越远，干季越长） |
| **海陆位置** | 可沿海可内陆，但**常位于大陆内部的高原/台地** | 如巴西高原、东非高原——海拔使温度降低，但不足以改变热带属性，同时加剧干湿季对比 |
| **洋流** | 影响因地而异 | 东非高原受印度洋季风影响（部分沿岸有暖流）；巴西高原受大西洋信风影响；墨西哥高原受加勒比海暖水影响 |
| **地形** | **高原/台地**常见 | 海拔800-1500m的高原 → 温度适中（避开了低地的酷热）→ 典型的热带草原景观 |

#### 典型地区
| 地区 | 具体位置 | 干季特征 |
|------|----------|----------|
| 巴西高原（Cerrado） | 南美洲中部10°-20°S | 南半球冬季（5-9月）为干季 |
| 东非高原（Serengeti） | 肯尼亚-坦桑尼亚1°-3°S | 双峰降水（ITCZ两次经过），但仍有两个干季 |
| 印度半岛内陆（德干高原） | 印度中部15°-20°N | 冬季干季显著，但夏季季风雨强烈 |
| 澳大利亚北部 | 10°-20°S | 夏季（12-3月）受西北季风（越赤道偏转）影响为雨季 |

#### Aw vs Am 的关键区分

| 对比维度 | Am（热带季风） | Aw（热带草原） |
|----------|---------------|---------------|
| 干季长度 | 1-3个月 | 3-6个月 |
| 年降水量 | 1500-3000mm+ | 600-1500mm |
| 形成机制 | 季风环流主导 | ITCZ季节移动主导 |
| 干燥期的控制系统 | 信风或大陆气团 | 副热带高压（下沉气流） |
| 典型位置 | 大陆沿海迎风坡 | 大陆内部高原 |

## 三、B类 干燥气候

### B类的根本成因：降水 < 潜在蒸散

干燥气候不一定是"不下雨"，而是**蒸发需求超过了降水供给**。这意味着B类气候可以出现在年降水不低但蒸发极强的地区（如部分热带草原边缘），也可以出现在降水极少的极端沙漠。

---

### 3.1 BWh — 热带/副热带沙漠气候（Hot Desert）

| 维度 | 典型特征 |
|------|----------|
| **柯本标准** | 年均温 ≥ 18°C；年降水量 < 蒸散阈值的50% |

#### 五大驱动因素（三条形成路径）

**路径A：副热带大陆西岸型（最典型，占全球沙漠面积最大）**

| 因素 | 典型组合 | 成因说明 |
|------|----------|----------|
| **纬度** | 15° - 30° N/S | 位于副热带高压带的核心纬度 |
| **大气环流** | 终年受**副热带高压**控制，盛行**下沉气流** | 下沉空气绝热增温 → 相对湿度下降 → 云量极少 → 日照极强 → 地面辐射冷却夜间温度 | 
| | 信风为**离岸风**（从陆地吹向海洋） | 无法从海洋携带水汽登陆 |
| **海陆位置** | 大陆**西岸** | 位于信风带的下风向（东-西走向的信风在到达西岸前已穿越大片陆地，失去水汽） |
| **洋流** | 沿岸为**寒流** | 加那利寒流（撒哈拉西岸）、本格拉寒流（纳米布）、秘鲁寒流（阿塔卡马）、加利福尼亚寒流（索诺拉）→ 近海面空气冷却 → 形成逆温层 → 抑制对流和降水 |
| **地形** | 通常地势平坦或为低矮高原 | 无地形抬升降水机制（与路径B不同） |

**典型地区**：撒哈拉沙漠、阿拉伯沙漠、澳大利亚西部沙漠、阿塔卡马沙漠、纳米布沙漠、索诺拉沙漠

**路径B：大陆内部雨影型**

| 因素 | 典型组合 | 成因说明 |
|------|----------|----------|
| **纬度** | 任何热带-温带纬度 | 不依赖纬度 |
| **大气环流** | 各方向风经过山脉后水汽已被耗尽 | 多方向来风均为干风 |
| **海陆位置** | **大陆腹地**，距海>1000-2000km | 所有方向的水汽均无法到达 |
| **洋流** | 无影响（太远） | — |
| **地形** | **四面环山/位于背风坡（雨影区）** | 周围山脉在迎风面截留所有水汽 → 内陆盆地极度干燥 |

**典型地区**：中亚沙漠（卡拉库姆、克孜勒库姆）、中国塔克拉玛干沙漠、美国大盆地沙漠

**路径C：极低降水+强烈蒸发的组合**

- 任何满足"蒸发 >> 降水"条件的地区，即使降水量不算极端低
- 例如：埃塞俄比亚达纳基尔洼地（年降水约100-200mm，但年均温>30°C → 蒸发极强 → 形成盐漠）

#### 成因机制链（以副热带西岸型BWh为例）
```
15-30°N/S → 副热带高压终年控制
    ↓                         ↓
下沉气流 → 绝热增温 → 湿度低    寒流 → 逆温层 → 对流抑制
    ↓                         ↓
           离岸信风不带来水汽
                ↓
   云量极少 + 日照极强 + 降水罕见（<50-100mm/yr）
                ↓
    结果：BWh 热带沙漠气候 → 地球上最干旱的气候类型
```

#### BWh的三个极致案例

| 沙漠 | 位置 | 极致之处 | 关键机制 |
|------|------|----------|----------|
| 阿塔卡马沙漠 | 智利-秘鲁 | 部分地区从未记录到降水 | 秘鲁寒流（世界最强寒流之一）+ 安第斯山雨影 + 副高 |
| 撒哈拉沙漠 | 北非 | 面积最大（约900万km²） | 副高 + 离岸东北信风 + 加那利寒流 + 东西宽阔大陆 |
| 卢特荒漠 | 伊朗 | 地面温度最高（曾记录70.7°C） | 内陆盆地 + 周围山脉阻挡所有水汽 |

### 3.2 BWk — 中纬度冷沙漠（Cold Desert）

| 维度 | 典型特征 |
|------|----------|
| **柯本标准** | 年均温 < 18°C；满足BW的干燥标准 |

#### 五大驱动因素

| 因素 | 典型组合 | 成因说明 |
|------|----------|----------|
| **纬度** | 35° - 55° N（主要在亚洲内陆，南半球中纬度陆地面积太小，BWk罕见） | 中纬度太阳辐射，冬季寒冷 |
| **大气环流** | 位于盛行西风带的**内陆终端** | 西风从大西洋出发，穿越数千公里欧亚大陆后水汽耗尽 → 到达中亚时已是干风；冬季受大陆冷高压控制，空气干燥 |
| **海陆位置** | **极端大陆性**，距任何海洋 > 2000km | 欧亚大陆中心是全球距海最远的地区 |
| **洋流** | 无任何影响 | — |
| **地形** | **四周山脉环绕的盆地** | 天山、帕米尔高原、阿尔泰山、青藏高原 → 从四面封锁水汽 → 中亚的塔克拉玛干、卡拉库姆沙漠 |

#### 典型地区
| 地区 | 位置 | 年降水 | 1月均温 |
|------|------|--------|---------|
| 塔克拉玛干沙漠 | 中国新疆，塔里木盆地 | <50mm | -5至-10°C |
| 戈壁沙漠 | 蒙古-中国内蒙古 | 50-150mm | -15至-25°C |
| 大盆地沙漠 | 美国内华达/犹他 | 150-250mm | 0至-5°C |
| 卡拉库姆沙漠 | 土库曼斯坦 | 100-150mm | 0至2°C |

#### BWh vs BWk 的区分逻辑

| 对比维度 | BWh | BWk |
|----------|-----|-----|
| 年均温 | ≥ 18°C | < 18°C |
| 纬度 | 15°-30° | 35°-55° |
| 主要成因 | 副高+寒流 | 大陆腹地+山脉封锁 |
| 冬季 | 温和（>10°C） | 寒冷（<0°C） |
| 典型位置 | 副热带大陆西岸 | 中纬度大陆中心 |

### 3.3 BSh / BSk — 半干旱草原/荒漠草原（Hot / Cold Semi-arid）

#### BSh（热带/副热带草原）

| 因素 | 典型组合 |
|------|----------|
| **纬度** | 15°-35°N/S（沙漠的边缘过渡带） |
| **大气环流** | 沙漠和湿润气候之间的**过渡区**：夏季偶有ITCZ或季风雨到达，冬季受副高或内陆高压影响 |
| **海陆位置** | 沙漠的外围，或内陆高原 |
| **洋流** | 较弱的寒流或过渡洋流 |
| **地形** | 常为沙漠边缘的**高平原或台地**（如非洲萨赫勒、巴西东北部Sertão、澳大利亚内陆边缘） |

#### BSk（中纬度寒冷草原）

| 因素 | 典型组合 |
|------|----------|
| **纬度** | 35°-55°N/S |
| **大气环流** | 西风带，但水汽量不足以维持湿润气候 |
| **海陆位置** | 大陆内部或高山雨影区 |
| **洋流** | 无 |
| **地形** | **雨影区高地平原/山间盆地**（如北美大平原西部、蒙古草原、巴塔哥尼亚） |

#### BSh / BSk 典型地区
| 气候 | 地区 | 降水量 | 景观 |
|------|------|--------|------|
| BSh | 非洲萨赫勒带（塞内加尔-苏丹） | 200-600mm | 稀树草原、灌丛 |
| BSh | 巴西东北部Sertão | 300-700mm | 卡廷加（Caatinga）旱生林 |
| BSh | 印度西北部（拉贾斯坦边缘） | 200-500mm | 塔尔沙漠边缘灌丛草原 |
| BSk | 北美大平原西部（High Plains） | 250-500mm | 短草草原 |
| BSk | 蒙古草原 | 200-350mm | 温带草原 |
| BSk | 巴塔哥尼亚 | 150-300mm | 寒冷荒漠草原 |
| BSk | 中亚哈萨克草原 | 200-400mm | 温带干旱草原 |

## 四、C类 暖温带/亚热带气候（最冷月均温：0°C ≤ T < 18°C）

C类气候的共有基础：位于中低纬度（25°-45°），冬季温和但不炎热，至少有一个月均温低于18°C但高于0°C。

---

### 4.1 Cfa — 亚热带湿润气候（Humid Subtropical, Without Dry Season）

| 维度 | 典型特征 |
|------|----------|
| **柯本标准** | 最冷月均温 0-18°C，最热月均温 ≥ 22°C（夏季炎热a）；全年降水均匀（无干季） |

#### 五大驱动因素

| 因素 | 典型组合 | 成因说明 |
|------|----------|----------|
| **纬度** | 25° - 35° N/S | 亚热带纬度，夏季炎热，冬季温和 |
| **大气环流** | 夏季：**西太平洋副热带高压（或其同类）**西缘 → 东南/南风从海洋带来湿热空气；冬季：中纬度气旋活动带来降水；全年有降水机制 | 关键在于**大陆东岸**的位置——夏季副高位于海洋上，其西部边缘的南风/东南风将海洋水汽输送到大陆东岸 |
| **海陆位置** | 大陆**东岸**中低纬度 | 位于副高的西侧（北半球），盛行向岸的暖湿气流 |
| **洋流** | 沿岸**暖流** | 北大西洋暖流/墨西哥湾暖流（美国东南部）、黑潮（日本南部、中国东部沿海）、巴西暖流（巴西南部-阿根廷北部）、东澳大利亚暖流 → 提供水汽和热量 |
| **地形** | 多为沿海平原/低丘 | 无明显山脉阻挡水汽进入（若有山地迎风坡则降水更大） |

#### 典型地区
| 地区 | 具体位置 | 关键特征 |
|------|----------|----------|
| 美国东南部 | 佛罗里达-佐治亚-南卡罗来纳-得克萨斯东部 | 墨西哥湾暖流 + 夏季墨西哥湾水汽 |
| 中国长江中下游 | 上海-南京-武汉-长沙 | 东亚季风体系，但冬季降水仍可观（春雨+梅雨） |
| 日本南部（太平洋侧） | 九州、四国、本州南部 | 黑潮暖流 + 夏季东南季风 |
| 巴西南部-阿根廷东北 | 南回归线附近至30°S | 巴西暖流 + 南大西洋副高西缘 |
| 澳大利亚东海岸中段 | 布里斯班-悉尼北部 | 东澳大利亚暖流 + 东南信风 |

#### Cfa 的降水均匀性来源

Cfa降水均匀 ≠ 各月降水量完全相等，而是指**没有明显的干季**（最干月降水量 > 30-40mm）。其降水来源的多样性：

- **夏季**：对流雨 + 热带气旋/台风
- **春秋**：锋面降水（冷暖空气交汇）
- **冬季**：中纬度气旋通过 → 带来降水（与同纬度西岸的地中海气候形成对比——地中海气候冬季多雨但夏季干旱）

---

### 4.2 Cfb / Cfc — 温带海洋性气候（Oceanic / Marine West Coast）

| 维度 | 典型特征 |
|------|----------|
| **柯本标准** | 最冷月均温 0-18°C；最热月均温 < 22°C（Cfb，夏季温暖）；若最热月 < 22°C且≥4个月≥10°C则为Cfc（夏季凉爽，亚极地海洋性） |

#### 五大驱动因素

| 因素 | 典型组合 | 成因说明 |
|------|----------|----------|
| **纬度** | 35° - 55° N/S（Cfb）；55° - 60° N/S（Cfc） | 中纬度——恰好位于盛行西风带的核心区域 |
| **大气环流** | 终年受**盛行西风**控制 + 频繁的**中纬度气旋** | **西风从海洋持续向陆地输送暖湿空气** → 这是Cfb形成的"命门"。冬季：气旋活跃带来阴雨天气；夏季：虽西风稍弱，但海洋蒸发仍提供足够水汽 → 降水较均匀 |
| **海陆位置** | 大陆**西岸**，濒临广阔的海洋 | **这是Cfb出现在西岸而非东岸的根本原因**——西风从海洋吹向陆地，大陆西岸才是西风的"上风向" |
| **洋流** | **强大暖流**是关键放大器 | **北大西洋暖流**（西欧）、阿拉斯加暖流（北美西北岸）、巴西暖流的南向延伸 + 西风漂流（智利南部）、东澳大利亚暖流的南向延伸（新西兰、塔斯马尼亚） |
| **地形** | 沿海山脉迎风坡 → 降水增强；若无山脉阻挡 → 海洋性深入内陆 | 西欧：无南北向山脉阻挡，海洋性影响远达波兰；北美西北海岸：海岸山脉迎风坡 → 温带雨林（>3000mm降水） |

#### 典型地区
| 地区 | 范围 | 降水特征 | 暖流 |
|------|------|----------|------|
| 西欧 | 英国-爱尔兰-法国西部-比荷卢-德国西北-丹麦 | 年降水600-1000mm，均匀分布 | 北大西洋暖流 |
| 北美西北海岸 | 阿拉斯加东南-不列颠哥伦比亚-华盛顿-俄勒冈海岸 | 2000-4000mm（迎风坡温带雨林） | 阿拉斯加暖流 |
| 智利南部 | 40°-55°S智利西岸 | 2000-5000mm（迎风坡） | 西风漂流（相对温和） |
| 新西兰 | 全国大部分地区 | 800-2000mm | 东澳大利亚暖流延伸 |
| 塔斯马尼亚 | 澳大利亚南部岛屿 | 600-1500mm | 东澳大利亚暖流延伸 |

#### Cfb 与 Cfa 的关键区分

| 对比 | Cfa（亚热带湿润） | Cfb（温带海洋性） |
|------|-----------------|-----------------|
| 大陆位置 | **东岸** | **西岸** |
| 环流 | 夏季副高，冬季气旋 | 终年西风 + 气旋 |
| 夏季气温 | 炎热（≥22°C） | 温暖/凉爽（<22°C） |
| 冬季气温 | 温和（常>5°C） | 温和（常>2°C） |
| 年温差 | 15-20°C | 8-15°C |
| 降水季节分配 | 夏季稍多（对流+台风） | 冬季稍多（气旋活跃）或均匀 |

#### 为什么西欧的Cfb能延伸到如此内陆？

这是全球气候中最特殊的地理现象之一。对比北美西岸：
- **北美**：落基山脉-海岸山脉南北纵列 → 西风携带的海洋水汽在海岸山脉西坡即被截留 → 海洋性气候仅限狭窄的沿海地带
- **欧洲**：阿尔卑斯山和喀尔巴阡山为**东西走向** → 不阻挡西风东进 → 海洋性影响直达波兰和捷克

这就是**地形（山脉走向）对气候区域范围的巨大修正作用**。

### 4.3 Csa / Csb — 地中海气候（Mediterranean / Dry Summer Subtropical）

| 维度 | 典型特征 |
|------|----------|
| **柯本标准** | 最冷月均温 0-18°C；最热月均温 ≥ 22°C（Csa，夏季炎热）或 < 22°C（Csb，夏季温暖）；**夏季最干月降水量 < 30mm，且夏季降水量 < 冬季最湿月的1/3** |

地中海气候是**唯一一个"夏干冬湿"的气候类型**——这使其在所有气候中独一无二。

#### 五大驱动因素

| 因素 | 典型组合 | 成因说明 |
|------|----------|----------|
| **纬度** | 30° - 40° N/S（大陆西岸） | **这是地中海气候的"黄金纬度"**——恰好处于副热带高压（夏）和西风带（冬）的年度摆动范围内 |
| **大气环流** | **副热带高压（夏季）+ 盛行西风（冬季）的交替控制** | 这是地中海气候形成的根本机制：**夏季**：副高北移/南移（取决于半球）→ 该纬度带被副高控制 → 下沉气流 → 晴朗、干燥、炎热；**冬季**：副高向赤道方向退缩 → 西风带南移 → 该纬度带被西风控制 → 中纬度气旋带来降水 → 温和多雨 |
| **海陆位置** | 大陆**西岸**，濒临广阔海洋 | 与Cfb类似，必须在大陆西岸——冬季西风才能从海洋带来水汽。若在大陆东岸，冬季盛行离岸风（或季风）→ 干燥 |
| **洋流** | 沿岸为**寒流**（与直觉相反！） | 加利福尼亚寒流（北美地中海气候区）、加那利寒流（地中海盆地西口）、本格拉寒流（南非开普）、西澳大利亚寒流 → **寒流是地中海气候夏季干燥的放大器**：寒流冷却近岸空气 → 形成逆温 → 抑制对流 → 强化夏季的干旱 |
| **地形** | 沿海山脉/丘陵常见 | 加利福尼亚海岸山脉、地中海北岸的阿尔卑斯山系支脉、智利海岸山脉 → 地形强化降水在冬季的集中性（西风在迎风坡抬升） |

#### 典型地区（全球五大分布区）

| 地区 | 范围 | 纬度 | Csa/Csb | 寒流 |
|------|------|------|---------|------|
| **地中海盆地**（名称来源地） | 南欧-北非-黎凡特沿岸 | 30°-45°N | Csa为主 | 加那利寒流（间接影响） |
| **加利福尼亚** | 美国加州沿海-墨西哥下加利福尼亚北部 | 32°-42°N | Csa/Csb均有 | 加利福尼亚寒流 |
| **智利中部** | 圣地亚哥-康塞普西翁沿岸 | 30°-38°S | Csa/Csb | 秘鲁寒流（洪堡寒流） |
| **南非开普地区** | 开普敦及周边沿海 | 33°-35°S | Csa/Csb | 本格拉寒流 |
| **澳大利亚西南/南部** | 珀斯-阿德莱德沿岸 | 30°-35°S | Csa/Csb | 西澳大利亚寒流 |

#### 成因机制链（地中海气候最经典的形成逻辑）
```
30-40°N/S 大陆西岸——位于副热带高压和西风带的"角力区"
    ↓
夏季：副高向极地扩张 → 控制该纬度 → 下沉 + 寒流逆温
    → 晴朗、干燥、日照极强 → 夏季干旱（降水 0-20mm/月）
    ↓
冬季：副高退缩 → 西风带南移 → 中纬度气旋频繁经过
    → 阴雨天气 → 降水集中于此（最湿月可达 100-200mm）
    ↓
结果：Csa/Csb 地中海气候 → "夏天是沙漠，冬天是温带海洋"
```

#### Csa vs Csb 的区分

| 对比 | Csa | Csb |
|------|-----|-----|
| 最热月均温 | ≥ 22°C | < 22°C |
| 地理位置 | 靠内陆或较低纬度（如罗马、雅典、珀斯） | 靠海岸或较高纬度，寒流影响更强（如旧金山、开普敦、圣地亚哥） |
| 典型城市 | 罗马、雅典、耶路撒冷、珀斯 | 旧金山、开普敦、波尔图 |

---

### 4.4 Cwa / Cwb / Cwc — 亚热带季风性/高地气候（Dry-Winter Subtropical / Subtropical Highland）

| 维度 | 典型特征 |
|------|----------|
| **柯本标准** | 最冷月均温 0-18°C；**冬季干燥**（最干月降水 < 最湿月的1/10）；Cwa：最热月≥22°C；Cwb：最热月<22°C（高原型）；Cwc：最热月<22°C且仅1-3个月≥10°C |

#### 五大驱动因素

| 因素 | Cwa（低地季风型） | Cwb（高地型） |
|------|-----------------|-------------|
| **纬度** | 20°-30° N/S | 10°-25° N/S（但海拔>1500m） |
| **大气环流** | **季风环流**：夏季受季风/ITCZ影响多雨，冬季受内陆高压/信风控制干燥 | 与Cwa同源，但高海拔使其夏季温度不足以达到Cwa的≥22°C阈值 |
| **海陆位置** | 大陆东岸或内陆 | 内陆高原/高地（如东非高原、安第斯山间谷地、墨西哥高原） |
| **洋流** | 东岸暖流提供夏季水汽 | 影响因地形阻隔而较弱 |
| **地形** | 多在沿海平原/丘陵 | **主要成因来自海拔**——如果该地区处于海平面，它将是Aw/Am；海拔将其温度拉低至C类范围 |

#### 典型地区
| 类型 | 地区 | 海拔 | 关键特征 |
|------|------|------|----------|
| Cwa | 中国华南内陆（郴州-桂林-贵阳） | 200-1000m | 东亚季风，但冬季较Cfa更干燥 |
| Cwa | 印度北部平原（德里-勒克瑙） | 200-300m | 南亚季风，冬季干燥显著 |
| Cwa | 非洲中南部高原（赞比亚-津巴布韦-安哥拉） | 1000-1500m | ITCZ夏季到达 → 雨季；冬季副高 → 干季 |
| Cwb | 墨西哥高原（墨西哥城） | 2200m | 海拔使热带变"永恒之春" |
| Cwb | 埃塞俄比亚高原（亚的斯亚贝巴） | 2400m | 同纬度低地是Aw，高原上成为Cwb |
| Cwb | 安第斯山中段谷地（波哥大、基多、库斯科） | 2600-3300m | 热带纬度的高海拔 → "无夏"的Cwb |

#### Cwa 与 Cfa 的区分

| 对比 | Cfa（亚热带湿润） | Cwa（冬干亚热带） |
|------|-----------------|-----------------|
| 干季 | 无显著干季 | 冬季明显干燥 |
| 位置 | 更靠近海岸（如上海、东京） | 更靠内陆或受内陆风控制（如重庆、贵阳、德里） |
| 机制 | 全年有多种降水来源 | 夏季降水占绝对主导（>70%） |

#### Cwb 的成因：海拔打败了纬度

这是地理学中**"垂直地带性推翻水平地带性"**的经典体现：

- 赤道附近的基多（0°S, 海拔2830m）→ Cwb（年均温约15°C）
- 若基多在海平面高度 → 将是Af（终年高温多雨的热带雨林）

海拔每升高1000m降温约6.5°C，这意味着**2800m的海拔可以将热带变为温带**。

## 五、D类 冷温带/大陆性气候（最冷月均温 < 0°C，最热月均温 > 10°C）

D类气候仅在北半球大面积发育（南半球中纬度陆地面积太小）。

---

### 5.1 Dfa / Dfb / Dwa / Dwb — 温带大陆性湿润气候（Humid Continental）

| 维度 | 典型特征 |
|------|----------|
| **柯本标准** | 最冷月均温 < 0°C；最热月均温 > 10°C；降水满足非干燥标准 |

#### 五大驱动因素（Dfa/Dfb 西风型 vs Dwa/Dwb 季风型）

| 因素 | Dfa/Dfb（北美/欧洲型） | Dwa/Dwb（东亚型） |
|------|----------------------|-------------------|
| **纬度** | 35°-55°N | 35°-50°N |
| **大气环流** | 全年有**西风带**经过 + 气旋降水；夏季叠加对流雨；冬季受极地大陆气团控制 | **季风环流**主导：夏季东南季风带来降水；冬季西伯利亚高压控制 → 干燥严寒 → 降水高度集中于夏季 |
| **海陆位置** | 大陆**内部或东岸** | 大陆**东岸**（欧亚大陆东缘） |
| **洋流** | Dfa（北美东岸）：墨西哥湾暖流提供水汽；Dfb（欧洲内陆）：无洋流直接影响 | 黑潮暖流（夏季水汽来源之一），但冬季受近岸寒流（中国沿岸流）和离岸冬季风控制 |
| **地形** | 多为平原/低丘；大平原（北美）和东欧平原 | 沿海平原 → 内陆山脉/高原（太行山、大兴安岭等作为内陆边界） |

#### Dfa / Dfb 与 Dwa / Dwb 的对比

| 对比 | Dfa/Dfb（美欧型） | Dwa/Dwb（东亚型） |
|------|-----------------|-----------------|
| 降水峰值 | 夏季（对流+气旋） | 夏季（季风雨，占比>60-70%） |
| 冬季降水 | 仍有气旋降水 | 极度干燥（受西伯利亚高压控制） |
| 冬季温度 | 北美中西部极端冷（-10至-20°C） | 同样极端（-10至-25°C） |
| 典型城市 | 芝加哥(Dfa)、莫斯科(Dfb)、华沙(Dfb) | 北京(Dwa)、沈阳(Dwa)、哈尔滨(Dwb) |
| 大陆位置 | 北美中东部、欧洲东部 | 东亚 |

#### 成因机制链（东亚Dwa/Dwb）
```
35-50°N 大陆东岸——深受季风环流控制

夏季（5-9月）： 
    亚洲低压（印度低压）+ 西太平洋副高 → 东南季风
    → 从太平洋带来暖湿空气 → 集中了全年 60-80% 的降水
    → 同时夏季炎热（最热月 22-28°C）→ 雨热同期

冬季（11-3月）：
    西伯利亚高压（蒙古高压）→ 西北季风
    → 从西伯利亚内陆带来极干冷空气 → 晴朗，极少降水
    → 最冷月 -5至-25°C → 年温差可达 30-45°C（极端大陆性）
    
结果：Dwa 温带季风气候 → 夏季炎热多雨，冬季严寒干燥
```

### 5.2 Dfc / Dfd / Dwc / Dwd — 亚寒带/亚北极气候（Subarctic / Boreal）

| 维度 | 典型特征 |
|------|----------|
| **柯本标准** | 最冷月均温 < 0°C（Dfd < -38°C）；仅 1-3 个月均温 ≥ 10°C；有至少一个月的均温 < -3°C |

#### 五大驱动因素

| 因素 | 典型组合 | 成因说明 |
|------|----------|----------|
| **纬度** | 50° - 70° N | 高纬度，太阳辐射弱，冬季漫长严寒 |
| **大气环流** | 冬季：**极地高压**或极地大陆气团控制 → 强烈辐射冷却 → 温度极低（西伯利亚东北部Verhoyansk 1月均温约-46°C）；夏季：极锋北移 → 气旋活动带来有限降水（年降水仅200-500mm但蒸发弱，仍可维持针叶林） | 关键：漫长严寒的冬季（6-8个月月均温<0°C）+ 短暂的凉爽夏季 |
| **海陆位置** | **大陆腹地**（西伯利亚、加拿大北部、阿拉斯加内陆） | 距海遥远 → 缺乏海洋调节 → 冬季温度极低（极端大陆性的极致） |
| **洋流** | 无影响 | 距离海洋太远，且高纬度洋流多为寒流 |
| **地形** | 广阔的平原/低高原（西伯利亚平原、加拿大地盾） | 平坦地形允许冷空气自由流动和积聚（冷空气堆积在盆地/谷地中使温度更低） |

#### Dfc（冬季极端但非最极端）vs Dfd（冬季极端严寒）

| 对比 | Dfc | Dfd |
|------|-----|-----|
| 最冷月均温 | -3°C 至 -38°C | < -38°C |
| 典型地区 | 西伯利亚大部分、加拿大育空-西北地区 | 西伯利亚东北部（上扬斯克、奥伊米亚康） |
| 记录极值 | — | 奥伊米亚康：-67.7°C（北半球有人居住的最低温记录） |

#### Dwd — 东亚亚寒带季风型
- 出现在中国东北北部、俄罗斯远东
- 夏季东南季风带来有限降水
- 冬季西伯利亚高压控制 → 极端严寒干燥
- 与Dfc的区别在降水格局：Dwd降水集中于夏季（季风），Dfc降水更均匀

### 5.3 Dsa / Dsb / Dsc — 地中海型大陆性气候（罕见）

这是D类中的"夏干"亚型，极为罕见（仅出现在伊朗高原、土耳其东部、美国西部山间的高海拔盆地等）。

其成因 = **地中海环流机制（夏季副高 + 冬季西风）+ 高海拔/高纬度（使冬季温度降至D类范围）**

## 六、E类 极地气候（最热月均温 < 10°C）

---

### 6.1 ET — 苔原气候（Tundra）

| 维度 | 典型特征 |
|------|----------|
| **柯本标准** | 0°C ≤ 最热月均温 < 10°C；至少有一个月均温 ≥ 0°C（与EF的区分） |

#### 五大驱动因素

| 因素 | 典型组合 | 成因说明 |
|------|----------|----------|
| **纬度** | 60° - 75° N/S | 接近极圈，太阳辐射极弱。冬季长达8-10个月，夏季短暂（仅1-3个月均温>0°C） |
| **大气环流** | 冬季：**极地高压**控制 → 辐射冷却 → 极寒（-20至-40°C）；夏季：极锋北移 → 偶有气旋影响 → 少量降水（年降水100-250mm）；**极地东风带**终年盛行 | 降水总量极少，但因蒸发极弱（低温）→ 仍呈湿润状态（多沼泽、湖泊） |
| **海陆位置** | 北冰洋沿岸（欧亚大陆北缘、北美大陆北缘、格陵兰沿岸） | 近海但海冰覆盖 → 海洋调节作用被海冰阻断 → 冬季仍极寒 |
| **洋流** | 多为寒流（东格陵兰寒流、拉布拉多寒流等），但北大西洋暖流的末端延伸使挪威北部沿岸的ET范围向北退缩 | 洋流决定了ET的纬度边界——挪威北岸受暖流影响，ET界线推进至70°N+；而拉布拉多受寒流影响，ET南推至55°N |
| **地形** | 多为低平的沿海平原 | 地表为**永久冻土**（Permafrost），夏季仅表层解冻（活动层约0.5-2m）→ 水分无法下渗 → 形成广泛沼泽和湖泊 |

#### 典型地区

| 地区 | 纬度范围 | 特征 |
|------|----------|------|
| 俄罗斯北冰洋沿岸（泰梅尔-楚科奇） | 70°-75°N | 广阔的苔原平原，永久冻土深厚 |
| 加拿大北部（努纳武特） | 60°-75°N | 受哈德逊湾冷水和拉布拉多寒流影响，苔原南限压低 |
| 阿拉斯加北部（北坡） | 68°-71°N | 普鲁度湾沿岸，石油开采区 |
| 冰岛沿岸（低地） | 63°-66°N | 纬度不算极高，但寒流（东格陵兰寒流分支）使冰岛大部分为ET |
| 南乔治亚岛、凯尔盖朗群岛 | 50°-55°S | 南半球中纬度但受南极冷水包围 → ET（海洋性苔原） |

#### 洋流对ET分布的南北推移

这是洋流对极地气候形成**空间范围修正**的最佳例子：

- **挪威北部（~70°N）**：受北大西洋暖流末端影响 → 最热月均温可>10°C → **不属于ET，而属于Dfc**（亚寒带）
- **拉布拉多半岛（~55°N）**：受拉布拉多寒流影响 → 最热月均温<10°C → **属于ET**（苔原南移至中纬度）
- 两地的纬度差了整整15°，但气候被洋流翻转——**暖流将温暖向北推，寒流将寒冷向南拉**

### 6.2 EF — 冰原气候（Ice Cap）

| 维度 | 典型特征 |
|------|----------|
| **柯本标准** | 所有月份均温 < 0°C（最热月均温 < 0°C）——终年冰封 |

#### 五大驱动因素

| 因素 | 典型组合 | 成因说明 |
|------|----------|----------|
| **纬度** | 70° - 90° N/S | 极地核心区，太阳辐射全年极弱（极夜长达半年） |
| **大气环流** | 终年**极地高压**控制 → 强烈而持续的下沉气流 → **下降风（katabatic wind）**自冰盖中心向外吹 | 空气极度寒冷干燥（比沙漠更干燥——南极内陆年降水<50mm水当量，堪称"极地冷沙漠"） |
| **海陆位置** | 被冰盖覆盖的大陆/岛屿 | 格陵兰（除沿海狭窄地带外）、南极洲（除南极半岛北部外）——均为连续的大冰盖 |
| **洋流** | 周围为寒冷洋流和/或海冰包围 | 南极绕极流（ACC）是世界上最强大的洋流，将温暖海水隔离在南极大陆之外 |
| **地形** | 海拔极高（特殊！） | **南极冰盖平均海拔约2300m，最高点超过4000m** → 海拔+高纬度的双重降温效应 → 南极东方站曾记录-89.2°C（地球表面最低温） |

#### 南极洲为什么比北极冷得多？

这是一个融合了**所有五大因素**的综合地理问题：

| 因素 | 北极 | 南极 |
|------|------|------|
| **纬度** | 90°N（北冰洋） | 90°S（南极大陆） |
| **海拔** | 海平面（北冰洋海冰） | **平均海拔2300m**（冰盖）→ 气温直减率约-15°C |
| **海陆** | 海洋被陆地包围（北冰洋） | **大陆被海洋包围**（南极洲）→ 极端大陆性 |
| **洋流** | 北大西洋暖流可进入北冰洋 | **南极绕极流（ACC）**将暖水完全隔离 → 是地球上最强烈的热屏障 |
| **大气** | 极地高压，但受气旋扰动较多 | 终年极地高压 → 稳定而强大的下降风 |
| **结果** | 最低约-53°C（北冰洋测量站） | **最低约-89.2°C**（东方站，1983年） |

## 七、H类 高地气候（Highland Climate）

虽然柯本原始分类中并未将H作为独立大类，但现代气候学中普遍将**高地**视为因地形而异的特殊气候。

### 高地气候的核心：地形推翻一切

H类没有自己的"气候特征"——高原/山地的气候**取决于它所在纬度的基准气候类型 + 海拔修正**。

| 纬度 | 基准气候（海平面） | 低坡 | 中坡 | 高坡 | 山顶 |
|------|-----------------|------|------|------|------|
| 赤道（0°） | Af（热带雨林） | Af | Am/Aw | Cwb | ET/H（乞力马扎罗山顶：冰原） |
| 副热带（30°N） | BWh（沙漠） | BWh | BSk | Csa/Dsa | ET |
| 温带（45°N） | Cfb/Dfb | Cfb/Dfb | Dfc | ET | EF（阿尔卑斯山顶：冰原） |

### H类气候的关键规则

1. **气温直减率**：海拔每升1000m，气温降低约6.5°C
2. **降水随海拔先增后减**：通常最大降水带在2000-3500m（取决于纬度），之上空气过于稀薄/干燥
3. **迎风坡 vs 背风坡**：降水和湿度差异极大
4. **"垂直地带谱"**：从山脚到山顶，可以经历从热带雨林到冰原的完整气候序列——水平方向需要跨越上万公里的气候变化，在垂直方向仅需几千米

### 高地气候的典型代表

| 地区 | 海拔范围 | 气候系列（从低到高） |
|------|----------|-------------------|
| 乞力马扎罗山（3°S, 坦桑尼亚） | 900-5895m | Af（山脚）→ Cwb（中坡）→ ET（高坡）→ EF（山顶冰盖） |
| 喜马拉雅南坡（尼泊尔） | 100-8848m | Am/Cwa（低山）→ Cwb（中山）→ Dfc（高山）→ ET（极高）→ EF（顶峰） |
| 安第斯山中段（秘鲁-玻利维亚） | 0-6500m | BWh（西坡低地沿海）→ BSk（西坡中段）→ Cwb（高原谷地）→ ET（高山）→ EF（峰顶） |

## 九、判定方法论：三步确定一个未知地区的气候类型

```
【第一步】看纬度 → 确定热量带大类（A/B/C/D/E）

【第二步】看大陆东西岸 + 盛行风向 → 确定降水格局
    ├── 大陆西岸 + 低纬度 → 副高/信风(离岸) → BWh (沙漠)
    ├── 大陆西岸 + 中纬度 → 西风 → Cfb (海洋性)
    ├── 大陆西岸 + 30-40° → 副高/西风交替 → Csa (地中海)
    ├── 大陆东岸 + 中低纬度 → 季风/向岸信风 → Cfa/Cwa (湿润/季风)
    ├── 大陆东岸 + 中高纬度 → 季风 → Dwa (温带季风)
    ├── 大陆内部 + 任何纬度 → 大陆性 → 偏向 B 或 D
    └── 极地纬度 → E (苔原/冰原)

【第三步】看地形修正 → 海拔 + 山脉走向
    ├── 高海拔 → 降温，可能改变大类 (如 Aw→Cwb)
    ├── 迎风坡 → 降水增强
    ├── 背风坡 → 雨影 → 可能形成局地沙漠
    └── 山脉走向 → 决定海洋影响的范围
```
