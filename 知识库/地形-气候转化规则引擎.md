# 地形-气候转化规则引擎

## 将用户绘制的地形特征转化为可量化的气候修正参数

---

## 总览：从地形输入到气候输出

```
用户绘制的地形图
    │
    ├── 栅格化 → 每个像元提取：
    │   ├── 海拔 h (m)
    │   ├── 坡度 θ (°)
    │   ├── 坡向 α (相对于盛行风向的角度)
    │   ├── 距最近水体距离 d_water (km)
    │   ├── 水体面积 A_water (km²)
    │   ├── 山脉高度 H_mtn、宽度 W_mtn、走向 φ_mtn
    │   └── 山脉连续性 C_mtn (0-1)
    │
    └── 输入规则引擎 → 输出气候参数：
        ├── 修正后年均温 T'
        ├── 修正后年降水 P'
        ├── 年温差 ΔT_annual
        ├── 大陆度指数 K
        ├── 季风强度指数 M
        └── 湿度修正 ΔRH
```

---

## 绪论：气候形成的全部主要驱动因素

一个地区的气候是以下所有因素协同作用的产物。在规则引擎中，每个因素都对应一个可量化的修正模块。

---

### A. 天文因素（第一性原理）

- **A1. 纬度**
  - **影响机制**：决定太阳辐射的年总量和季节分配。纬度越低 → 太阳高度角越大 → 单位面积接收辐射越多 → 年均温越高；纬度越高 → 辐射的季节差异越大 → 年温差越大
  - **量化参数**：太阳辐射年总量 (W/m²)、天文可照时数 (h)、太阳高度角 (°)
  - **对应模块**：基准温度查表、年温差基准
  - **相关气候带**：热带 (0°-23.5°) / 温带 (23.5°-66.5°) / 寒带 (66.5°-90°)

---

### B. 大气环流因素（热量与水分的全球再分配）

- **B1. 三圈环流（Hadley / Ferrel / Polar Cell）**
  - **影响机制**：将赤道盈余热量向极地输送，同时决定了地球上的气压带和风带分布，是气候形成的最核心动力机制
  - **量化参数**：经圈环流强度指数、环流宽度 (°纬度)
  - **对应气候类型**：热带雨林 (Hadley上升支) / 副热带沙漠 (Hadley下沉支) / 温带海洋性 (Ferrel西风)

- **B2. 气压带**
  - **B2a. 赤道低压带 (ITCZ)**
    - **影响机制**：终年或季节性强烈上升气流 → 对流雨旺盛 → 形成热带雨林/热带季风/热带草原气候的雨季
    - **量化参数**：ITCZ 年均纬度位置、季节摆动幅度 (°纬度)
  - **B2b. 副热带高压带 (~30°)**
    - **影响机制**：终年或夏季盛行下沉气流 → 绝热增温、空气干燥 → 形成副热带沙漠和地中海气候的夏季干旱
    - **量化参数**：副高脊线纬度、强度指数 (hPa)
  - **B2c. 副极地低压带 (~60°)**
    - **影响机制**：冷暖空气交汇、极锋活跃 → 中纬度气旋频繁 → 带来大量降水
    - **量化参数**：极锋平均纬度、气旋频率 (次/年)
  - **B2d. 极地高压 (~90°)**
    - **影响机制**：终年寒冷收缩下沉 → 空气极度干燥 → 极地冷沙漠 + 下降风
    - **量化参数**：极地高压强度 (hPa)、下降风风速 (m/s)

- **B3. 风带（行星风系）**
  - **B3a. 信风带 (Trade Winds, 0°-30°)**
    - **影响机制**：向岸信风 → 从海洋带来水汽 → 大陆东岸湿润（如巴西东南沿海）；离岸信风 → 从陆地吹向海洋 → 大陆西岸干燥（如撒哈拉西岸）
    - **量化参数**：信风方向 (°)、信风风速 (m/s)、水汽通量 (g·m/kg·s)
  - **B3b. 盛行西风带 (Westerlies, 30°-60°)**
    - **影响机制**：从海洋吹向大陆西岸 → 西欧、北美西北沿海终年湿润；大陆东岸为离岸风 → 降水偏少
    - **量化参数**：西风风速 (m/s)、西风带南北摆动幅度 (°纬度)
  - **B3c. 极地东风带 (Polar Easterlies, 60°-90°)**
    - **影响机制**：从极地吹向副极地 → 携带极地干冷空气 → 加剧高纬度大陆东岸的严寒
    - **量化参数**：极地东风风速 (m/s)、频率 (%)

- **B4. 季风环流**
  - **B4a. 形成条件**
    - **影响机制**：巨大的海陆热力差异（夏季大陆为热低压、冬季为冷高压）+ 合适的纬度范围 (5°-50°) + 地形不封锁气流通道 → 产生季节性风向反转的季风环流
    - **量化参数**：夏季海陆温差 (°C)、冬季海陆温差 (°C)、季风强度指数 M (0-100)
  - **B4b. 东亚季风**
    - **影响机制**：夏季东南季风从太平洋带来水汽 → 雨热同期；冬季西北季风从西伯利亚带来干冷空气 → 寒冷干燥
    - **量化参数**：夏季风建立日期、冬季风强度、季风雨带位置
  - **B4c. 南亚季风**
    - **影响机制**：夏季西南季风（越赤道气流偏转）从印度洋带来巨量水汽 → 暴雨；受喜马拉雅山阻挡 → 南麓极端降水
    - **量化参数**：越赤道气流强度、季风爆发日期、季风槽位置
  - **B4d. 其他季风系统**
    - **西非季风**：西南风从几内亚湾带来水汽 → 萨赫勒带的雨季
    - **澳大利亚北部季风**：夏季西北风（越赤道偏转）→ 北部雨季
    - **北美季风**：墨西哥高原夏季季风 → 美国西南部夏季降水
  - **B4e. 高原季风**
    - **影响机制**：青藏高原夏季为热源 → 加强上升气流 → 强化亚洲季风；冬季为冷源 → 加强下沉气流

---

### C. 海陆分布因素（水汽来源与热力差异）

- **C1. 距海远近**
  - **影响机制**：距海越远 → 海洋调节作用越弱 → 年温差越大（大陆性增强）、降水量越少（水汽耗尽）。距海越近 → 海洋调节越强 → 年温差缩小、湿度较高
  - **量化参数**：有效海洋距离 d_effective (km)、大陆度指数 K (Gorczyński公式)
  - **对应模块**：大陆度/海洋度判定

- **C2. 相对于盛行风向的海岸位置**
  - **C2a. 大陆西岸（西风带）**
    - **影响机制**：位于西风的上风向 → 终年获得海洋水汽 → 形成温带海洋性气候 (Cfb/Cfc)
  - **C2b. 大陆西岸（信风带 / 副热带）**
    - **影响机制**：信风为离岸风 + 副高控制 + 寒流 → 极端干旱 → 形成副热带沙漠 (BWh)
  - **C2c. 大陆西岸（中低纬度过渡带）**
    - **影响机制**：夏季受副高控制（干燥）、冬季受西风控制（湿润）→ 夏干冬湿 → 形成地中海气候 (Csa/Csb)
  - **C2d. 大陆东岸（低中纬度）**
    - **影响机制**：副高西缘带来向岸暖湿气流 / 季风环流 → 夏季多雨 → 形成亚热带湿润 (Cfa) 或季风气候 (Cwa/Dwa)
  - **量化参数**：盛行风向 (°)、海岸线方向 (°)、水汽来源方位

- **C3. 洋流**
  - **C3a. 暖流**
    - **影响机制**：增温增湿 → 暖流流经的沿海地区冬季温度比同纬度高 10-15°C（如北大西洋暖流与西欧）→ 可为 Cfb 和 Cfc 气候的维持提供关键热量
    - **量化参数**：暖流表层温度 (°C)、暖流宽度与流量 (Sv = 10⁶m³/s)
    - **典型案例**：北大西洋暖流、黑潮、巴西暖流、莫桑比克暖流
  - **C3b. 寒流**
    - **影响机制**：降温减湿 → 近岸空气冷却形成逆温层 → 抑制对流和降水 → 加剧沿岸沙漠的干旱（如秘鲁寒流与阿塔卡马沙漠）→ 也是地中海气候夏季干旱的放大器
    - **量化参数**：寒流表层温度 (°C)、逆温层高度 (m)
    - **典型案例**：秘鲁寒流 (洪堡寒流)、本格拉寒流、加利福尼亚寒流、加那利寒流
  - **C3c. 上升流**
    - **影响机制**：离岸风将表层海水吹离 → 深层冷水上涌 → 进一步冷却近海空气 → 强化沿岸干旱；但带来丰富营养盐 → 世界著名渔场
    - **量化参数**：上升流速度 (m/day)、营养盐浓度

- **C4. 大陆面积与形状**
  - **影响机制**：大陆面积越大 → 内陆距海越远 → 极端大陆性越强；大陆东西宽度越窄 → 海洋影响可从两侧深入 → 大陆性减弱
  - **量化参数**：大陆东西宽度 (km)、南北长度 (km)、大陆纵深 (km)
  - **典型案例**：欧亚大陆（最大 → 极端大陆性最强）vs 南美洲（较窄 → 安第斯山以东仍有一定大陆性）

### D. 地形因素（非地带性的最终修正符）

- **D1. 海拔**
  - **D1a. 气温直减率（基本效应）**
    - **影响机制**：海拔每升高 100m → 气温下降约 0.65°C（全球平均）→ 同一纬度从山脚到山顶可经历多个气候带
    - **量化参数**：气温直减率 Γ (°C/100m)、海拔 h (m)
    - **对应模块**：海拔-温度修正
  - **D1b. 高原热力效应**
    - **影响机制**：高海拔大面积高原（>3000m, >10万km²）→ 夏季感热加热强烈 → 实际温度比直减率预测高 3-8°C → 形成高原季风（青藏高原效应）
    - **量化参数**：高原面积 (km²)、高原平均海拔 (m)
  - **D1c. 逆温层（山谷冷湖）**
    - **影响机制**：封闭盆地/山谷 + 冬季 + 晴朗无风 → 冷空气沿坡面下沉至谷底堆积 → 谷底比坡面更冷（打破正常直减率）
    - **量化参数**：盆地封闭度 (0-1)、逆温强度 (°C/100m)
  - **D1d. 雪线**
    - **影响机制**：海拔超过雪线 → 地表被永久积雪/冰川覆盖 → 高反照率 (0.7-0.9) 反射大部分太阳辐射 → 进一步降温 2-5°C
    - **量化参数**：雪线海拔 (m，随纬度变化)

- **D2. 山脉走向**
  - **D2a. 与盛行风向的夹角**
    - **影响机制**：山脉走向与盛行风向垂直 → 最大抬升效率 → 强雨影和焚风；山脉走向与风向平行 → 无地形效应 → 气流可顺畅通过
    - **量化参数**：抬升效率 η = sin(风向与山脉法线的夹角)
    - **对应模块**：山脉气象效应
  - **D2b. 走向对气候区域范围的影响**
    - **影响机制**：南北走向的山脉（如北美科迪勒拉山系）→ 阻挡西风带的海洋水汽向东深入 → 海洋性气候仅限于狭窄沿海；东西走向的山脉（如欧洲阿尔卑斯山）→ 不阻挡西风东进 → 海洋性气候可深入内陆上千公里
    - **量化参数**：山脉走向角 φ (°)
    - **典型案例**：西欧（无南北向屏障 → Cfb 远达波兰）vs 北美西岸（海岸山脉阻挡 → Cfb仅限沿海50-100km）

- **D3. 山脉高度**
  - **D3a. 对降水的影响**
    - **影响机制**：山脉高度 H > 抬升凝结高度 LCL → 空气达到饱和 → 形成地形雨 → 迎风坡降水增强；H < LCL → 仅产生地形云，无显著降水增强
    - **量化参数**：LCL (m) = 125×(T_surface - T_dewpoint)、最大降水高度 H_optimal (2000-3500m)
    - **对应模块**：迎风坡降水增强
  - **D3b. 对雨影强度的影响**
    - **影响机制**：山脉越高 → 背风坡降水衰减越慢（衰减距离 D 越大）→ 雨影区延伸越远。H=1000m → D≈15km；H>4000m → D≈120km
    - **对应模块**：雨影效应
  - **D3c. 对焚风强度的影响**
    - **影响机制**：山脉高度 H - LCL 越大 → 空气下沉距离越大 → 干绝热增温越强 → 焚风升温越显著。ΔT_foehn ≈ 0.5×(H-LCL)/100 (°C)
    - **对应模块**：焚风效应

- **D4. 山脉连续性**
  - **D4a. 连续山脉**
    - **影响机制**：连续无缺口的山脉 (C≈1.0) → 全方位阻挡气流 → 两侧气候差异极大 → 山脉即为鲜明气候分界线
    - **典型案例**：喜马拉雅山（阻挡南亚季风深入青藏高原）、秦岭（中国南北气候分界线）
  - **D4b. 有缺口的山脉**
    - **影响机制**：缺口/河谷形成"水汽通道"→ 湿气可从迎风坡通过缺口到达背风坡 → 山脉效应的强度和范围被削弱。缺口比例越大 → 整体效应越弱 (η_effective = η×C; C<0.5 时效应显著减弱)
    - **量化参数**：山脉连续性 C = 1 - Σ(缺口宽度)/山脊总长度
    - **典型案例**：长江三峡（四川盆地水汽通道）、恒河平原缺口（季风深入印度北部）

- **D5. 坡度**
  - **D5a. 迎风坡坡度**
    - **影响机制**：坡度越大 → 气流抬升越剧烈 → 上升速度越快 → 冷却和凝结速率越高 → 降水量越大（但存在上限，过陡反而使降水带变窄）
    - **量化参数**：迎风坡平均坡度 θ_windward (°)
  - **D5b. 背风坡坡度**
    - **影响机制**：背风坡越陡峭 → 下沉越快 → 焚风升温越集中 → 在距山脊较近距离内即可完成剧烈升温
    - **量化参数**：背风坡平均坡度 θ_leeward (°)

---

### E. 地表覆盖因素（能量平衡与水分循环的交互面）

- **E1. 陆地表面类型（反照率效应）**
  - **E1a. 森林**
    - **影响机制**：低反照率 (0.08-0.15) → 吸收更多太阳辐射 → 但蒸腾作用强烈 → 将大量热量转化为潜热（降温效应）→ 净效应：降低日间最高温、增加空气湿度 → 形成局地林内微气候
    - **量化参数**：反照率 α、叶面积指数 LAI、蒸散率 ET
  - **E1b. 草地/草原**
    - **影响机制**：中等反照率 (0.16-0.26) → 蒸腾中等 → 季节性变化显著（雨季绿、旱季枯黄）
    - **量化参数**：反照率 α、植被覆盖率 (%)
  - **E1c. 裸地/沙漠**
    - **影响机制**：高反照率 (0.25-0.40) → 反射大量太阳辐射 → 但无蒸腾降温 → 地表白天极热、夜间辐射冷却极快 → 日温差极大 (可达 30-50°C)
    - **量化参数**：反照率 α、地表热容量
  - **E1d. 冰雪**
    - **影响机制**：极高反照率 (0.70-0.90) → 反射绝大部分太阳辐射 → 地表温度远低于同纬度非冰雪区 → 产生正反馈：更冷 → 更多冰雪 → 更冷（冰雪-反照率反馈）
    - **量化参数**：反照率 α (0.70-0.90)、积雪覆盖率 (%)
  - **E1e. 城市/人工地表**
    - **影响机制**：低反照率 (0.10-0.20) + 高蓄热 + 人为热源 → 热岛效应 → 城市年均温比郊区高 1-3°C，夜间温差可达 5-10°C
    - **量化参数**：城市热岛强度 ΔT_uhi (°C)、不透水面比例 (%)

- **E2. 水体表面类型**
  - **E2a. 海洋**
    - **影响机制**：极低反照率 (0.05-0.10) → 吸收大量太阳辐射 → 巨大热容量使温度变化缓慢 → 作为巨大的热量和水分储存库 → 调节沿岸气候（冬暖夏凉、增加湿度）
    - **量化参数**：海表温度 SST (°C)、蒸发率 (mm/yr)
  - **E2b. 大型湖泊 (>1000km²)**
    - **影响机制**：类似海洋但影响范围较小 → 下风向 30-80km 内温度受调节 → 冬季湖效应降水（冷空气越过暖水面→剧烈降雪）
    - **量化参数**：湖泊面积 A (km²)、深度 Z (m)、影响范围 D_influence (km)
  - **E2c. 中小型湖泊/水库**
    - **影响机制**：影响范围有限 (0.5-10km) → 仅产生局地微气候调节 → 小型水体更多影响局地湿度和雾频率
    - **量化参数**：水体面积 A (km²)

- **E3. 植被的蒸散作用**
  - **E3a. 蒸散的水分循环效应**
    - **影响机制**：植被蒸散将土壤水分返回大气 → 增加近地面空气湿度 → 为对流降水提供水汽来源 → 植被茂盛区（如热带雨林）降水中的相当比例来自局地蒸散（"水分再循环"）→ 大规模毁林可能导致降水减少
    - **量化参数**：蒸散量 ET (mm/yr)、水分再循环率 (%)
  - **E3b. 植被的粗糙度效应**
    - **影响机制**：森林冠层增加地表粗糙度 → 增加湍流混合 → 影响近地面风速、温度和湿度廓线 → 森林可使风速降低 30-70%
    - **量化参数**：粗糙度长度 z₀ (m)

- **E4. 土壤湿度**
  - **E4a. 湿润土壤**
    - **影响机制**：土壤含水量高 → 更多能量用于蒸发而非加热地表 → 降低日间最高温；夜间蒸发减弱 → 降温幅度较小 → 日温差缩小
    - **量化参数**：土壤湿度 (m³/m³)、蒸发比 (EF)
  - **E4b. 干燥土壤**
    - **影响机制**：无蒸发降温 → 地表温度完全由辐射平衡决定 → 白天极热、夜间辐射冷却剧烈 → 日温差大 → 易形成干旱-高温正反馈（越干越热、越热越干）
    - **量化参数**：土壤湿度 (m³/m³)

## 第零部　非地形基准规则（纬度 · 大气环流 · 洋流 · 季风）

地形是气候的修正符——但在应用地形修正之前，必须先确定一个地区的**基准气候参数**。基准气候由纬度、大气环流、洋流和季风环流共同决定。以下规则引擎按"先定基准，再叠地形"的顺序组织。

### 零.1　纬度 → 太阳辐射 → 基准温度

#### 原理

纬度决定到达大气顶层的太阳辐射量，是气候系统的第一推动力。

```
年均太阳辐射 I(φ) = S₀/π × [1 + 0.033×cos(2π×n/365)] × cos(φ) × f(φ)

简化（年均）：
  I_annual(φ) ≈ 300 × cos(φ − 5°)  (W/m², 日平均)

其中 φ = 纬度 (°), S₀ = 1361 W/m² (太阳常数)
```

#### 基准温度查表与公式

```
规则0.1（纬度→基准年均温）：
  GIVEN 纬度 φ (°)
  THEN:
    // 海平面年均温（无洋流/无特殊环流修正）
    T_base(φ) = 27.0 − 0.68 × |φ|  (适用于 |φ| ≤ 55°)
    T_base(φ) = 27.0 − 0.68×55 − 1.2 × (|φ| − 55)  (适用于 |φ| > 55°)
    
    // 或分段查表

规则0.1b（纬度→年温差估算）：
  GIVEN 纬度 φ
       距海距离 d (km)
  THEN:
    A_base = 3 + 0.62 × |φ|  (海洋性基准年温差)
    A_continental = 3 + 1.35 × |φ|  (大陆性基准年温差)
    
    实际 A = A_oceanic + (A_continental − A_oceanic) × (1 − e^(−d/500))
```

### 零.2　大气环流规则（气压带 · 风带 · 降水带）

#### 原理

大气环流决定了降水在全球的带状分布。三圈环流在南北半球各形成四个气压带和三个风带。**气压带的位置随季节南北移动约 5°-10°**。

```
气压带理想化分布（年均位置）：

  90°N —— 极地高压 (Polar High)           → ↓ 下沉，干燥
  60°N —— 副极地低压 (Subpolar Low)       → ↑ 上升，多气旋雨
  30°N —— 副热带高压 (Subtropical High)   → ↓ 下沉，干燥
   0°  —— 赤道低压 / ITCZ                  → ↑ 上升，多对流雨
  30°S —— 副热带高压
  60°S —— 副极地低压
  90°S —— 极地高压
```

#### 气压带季节移动规则

```
规则0.2（气压带位置计算）：
  GIVEN 当前月份 M (1-12)
       纬度 φ
  THEN:
    // ITCZ 纬度
    ITCZ_lat = 0 + 8 × sin(2π × (M − 4) / 12)  (°)
    // 7-8月 ITCZ 最北 (~8°N), 1月最南 (~−8°S)
    // 4月和10月 ITCZ 在赤道附近
    
    // 副热带高压脊线纬度（北半球）
    STH_N_lat = 28 + 6 × sin(2π × (M − 4) / 12)  (°)
    // 夏季北移至~34°N, 冬季南退至~22°N
    
    // 副极地低压纬度（北半球）
    SPL_N_lat = 58 + 8 × sin(2π × (M − 4) / 12)  (°)
    // 夏季~66°N, 冬季~50°N

规则0.2b（该地当前受哪个气压带控制）：
  GIVEN 纬度 φ, 月份 M
  THEN:
    最近气压带 = argmin(|φ − 各气压带当前纬度|)
    
    IF 最近气压带 = ITCZ OR 副极地低压 THEN
      盛行上升气流 → 降水增强 (P_multiplier = 1.5~3.0)
    IF 最近气压带 = 副热带高压 OR 极地高压 THEN
      盛行下沉气流 → 降水抑制 (P_multiplier = 0.05~0.30)
```

#### 风带与降水关系

```
规则0.2c（风带判定）：
  GIVEN 纬度 φ
  THEN:
    IF |φ| < 5° THEN wind_belt = "赤道无风带/ITCZ" → 上升气流 → 对流雨
    
    IF 5° ≤ |φ| < 28° THEN wind_belt = "信风带"
      → 北半球：东北信风；南半球：东南信风
      → 大陆东岸：向岸风 → 湿润
      → 大陆西岸：离岸风 → 干燥
    
    IF 28° ≤ |φ| < 32° THEN wind_belt = "副热带无风带/马纬度"
      → 下沉气流 → 干燥，风力弱
    
    IF 32° ≤ |φ| < 58° THEN wind_belt = "盛行西风带"
      → 北半球：西南风；南半球：西北风
      → 大陆西岸：向岸风 → 湿润
      → 大陆东岸：离岸风 → 干燥
    
    IF |φ| ≥ 58° THEN wind_belt = "极地东风带"
      → 从极地吹向副极地
      → 干冷

  GIVEN 风带, 海岸位置, 距海距离 d
  THEN:
    IF wind_belt = "信风带" THEN
      IF coast = "东岸" AND d < 300 THEN P_mult = 1.2 ~ 2.0  // 向岸信风
      IF coast = "西岸" THEN P_mult = 0.1 ~ 0.3              // 离岸信风
    IF wind_belt = "西风带" THEN
      IF coast = "西岸" AND d < 500 THEN P_mult = 1.2 ~ 2.5  // 向岸西风
      IF coast = "东岸" THEN P_mult = 0.3 ~ 0.6              // 离岸西风
    IF wind_belt = "ITCZ" THEN P_mult = 1.5 ~ 3.0
    IF wind_belt = "副热带无风带" THEN P_mult = 0.05 ~ 0.20
    IF wind_belt = "极地东风带" THEN P_mult = 0.10 ~ 0.30
    IF wind_belt = "副极地低压" THEN P_mult = 1.3 ~ 1.8       // 极锋气旋降水
```

### 零.3　洋流修正规则

#### 原理

洋流对沿岸气候产生**温度和水分的双重调节**。暖流使沿岸冬季温度比同纬度高 10-15°C；寒流使沿岸形成逆温层，强化干旱。

#### 暖流修正规则

```
规则0.3（暖流温度修正）：
  GIVEN 洋流表层温度 SST (°C)
       同纬度无洋流基准温度 T_base (°C)
       距海岸距离 d_coast (km)
       盛行风向（是否从洋面吹向陆地）
  THEN:
    ΔT_anomaly = SST − T_expected(φ)
    // 暖流异常 = 实际海温 − 同纬度平均海温
    
    IF ΔT_anomaly > 3°C THEN  // 显著暖流
      沿岸温度修正：
        ΔT_coastal = ΔT_anomaly × e^(−d_coast / D_warm)
        D_warm ≈ 200 km  (暖流影响向内陆的衰减距离)
        
        最大冬季加温 = min(15, ΔT_anomaly × 1.2)  (°C)
        // 冬季效应更大（海洋释放热量）
        
      沿岸降水修正：
        ΔP_coastal = +15% ~ +40%  
        // 暖流增加蒸发 → 增加向岸风携带的水汽

暖流影响范围分级：
  ΔT_anomaly > 8°C → 极强暖流（如北大西洋暖流、黑潮） → 冬季升温 10-15°C
  ΔT_anomaly 5-8°C → 强暖流（如巴西暖流） → 冬季升温 5-10°C
  ΔT_anomaly 2-5°C → 中等暖流 → 冬季升温 2-5°C
  ΔT_anomaly < 2°C → 弱暖流或无暖流 → 无显著修正
```

#### 寒流修正规则

```
规则0.3b（寒流效应）：
  GIVEN 洋流表层温度 SST (°C)
       同纬度平均海温 T_expected (°C)
       距海岸距离 d_coast (km)
  THEN:
    ΔT_anomaly = SST − T_expected(φ)  // 负值
    
    IF ΔT_anomaly < −3°C THEN  // 显著寒流
      沿岸温度修正：
        ΔT_coastal = ΔT_anomaly × e^(−d_coast / D_cold)
        D_cold ≈ 50 km  (寒流影响限于狭窄沿海带)
        
      逆温层效应：
        IF 纬度在 15°-35° AND 大陆西岸 THEN
          逆温层强度 = |ΔT_anomaly| × 0.3  (°C/100m)
          P_suppression = −40% ~ −80%
          // 逆温抑制对流 → 极端干旱
          fog_frequency = "高" (平流雾，尤其夏季)
        
      最大夏季降温 = max(8, |ΔT_anomaly| × 0.8)  (°C)
      // 夏季降温显著（冷水抑制气温升高）

寒流影响范围分级：
  ΔT_anomaly < −8°C → 极强寒流（如秘鲁寒流） → 降温 6-8°C，降水减少 60-80%
  ΔT_anomaly −5 ~ −8°C → 强寒流（如本格拉寒流） → 降温 4-6°C，降水减少 40-60%
  ΔT_anomaly −2 ~ −5°C → 中等寒流（如加利福尼亚寒流） → 降温 1-3°C
  ΔT_anomaly > −2°C → 弱寒流或无寒流 → 无显著修正
```

### 零.4　季风环流规则

#### 原理

季风是大尺度海陆热力差异驱动的季节性风向反转。季风的存在与否、强度大小取决于**大陆规模、海陆温差、纬度位置和地形开放度**。

#### 季风形成判定

```
规则0.4（季风存在性判定）：
  GIVEN 纬度 φ
       大陆面积 A_land (km²)
       相邻海洋面积 A_ocean (km²)
       夏季海陆温差 ΔT_summer_land_sea (°C)
       距海距离 d (km)
       沿海地形阻力 R_terrain (0-1)
  THEN:
    // 必要条件检查
    cond_1 = (5 ≤ |φ| ≤ 50)               // 合适的纬度
    cond_2 = (A_land > 2×10⁶ km²)           // 大陆足够大
    cond_3 = (A_ocean > 5×10⁶ km²)          // 相邻暖洋足够大
    cond_4 = (ΔT_summer_land_sea > 8°C)     // 夏季海陆温差足够大
    
    IF cond_1 AND cond_2 AND cond_3 AND cond_4 THEN
      该地为季风区
      计算季风强度 M
    ELSE
      非季风区
      → 降水的季节性由 ITCZ 移动、西风气旋、或副高交替控制
```

#### 季风强度计算

```
规则0.4b（季风强度公式）：
  GIVEN 海陆夏季温差 ΔT_summer (°C)
       大陆纵深 D_land (km)
       海洋纵深 D_ocean (km)
       地形阻力 R_terrain (0-1)
  THEN:
    M = 15 × ΔT_summer × f_land × f_ocean × f_terrain
    
    其中:
      f_land = 1 + 0.3 × min(D_land, 5000) / 1000
      f_ocean = 1 + 0.2 × min(D_ocean, 5000) / 1000
      f_terrain = 1 − 0.8 × R_terrain
    
    M 值范围：0 ~ 120

季风强度分级：
  M ≥ 80  → 极强季风 → 夏季降水占全年 > 80% → 南亚(印度西海岸、孟加拉)
  M 60-79 → 强季风   → 夏季降水占 65-80%   → 东亚(中国东部、日本南部)
  M 40-59 → 中等季风 → 夏季降水占 50-65%   → 西非、澳大利亚北部
  M 20-39 → 弱季风   → 夏季稍多但不显著    → 美国东南部(副高西缘型)
  M < 20  → 非季风区 → 降水均匀或由其他机制决定
```

#### 冬季风与夏季风

```
规则0.4c（季风季节性表现）：
  IF M ≥ 40 THEN:
    // 夏季风（5-9月 北半球 / 11-3月 南半球）
    夏季风向 = 
      IF 东亚季风区 THEN 东南风
      IF 南亚季风区 THEN 西南风（SE信风越过赤道偏转）
      IF 西非季风区 THEN 西南风
    
    夏季降水占比 = 
      M < 40 → 35-50%
      M 40-59 → 50-65%
      M 60-79 → 65-80%
      M ≥ 80 → > 80%
    
    // 冬季风（11-3月 北半球 / 5-9月 南半球）
    冬季风向 = 与夏季相反
    冬季降水 = 极少（< 全年10%）
    
    // 雨热同期是季风气候的根本特征
    // 最热月与最湿月高度重合
```

#### 季风深入内陆的距离

```
规则0.4d（季风穿透距离）：
  GIVEN 季风强度 M (0-120)
       沿海地形阻力 R_terrain (0-1)
       是否有平原/河谷缺口
  THEN:
    基础穿透距离 D_0 = M × 20  (km)
    实际穿透距离 D_penetration = D_0 × (1 − R_terrain) × f_gap
    
    IF 存在大型平原缺口（宽度 > 50km）THEN
      f_gap = 1.2 ~ 1.5  // 缺口增强穿透
    ELSE
      f_gap = 1.0
    
    降水沿程衰减：
      P(x) = P_coast × e^(−x / (D_penetration × 0.5))
      其中 x = 距海岸的距离 (km)
    
    参考值：
      南亚季风(恒河平原缺口)    → D_penetration ≈ 2500km（到达青藏高原南麓）
      东亚季风(长江中下游平原)  → D_penetration ≈ 1500km（到达四川盆地东缘）
      东亚季风(华北平原)       → D_penetration ≈ 800km（到达太行山）
```

| 典型气候类型 | Cwa, Cfa, Dwa, Dwb, Am | Cfb, Csa, Af, BWh, Dfa |

## 一、海拔-温度修正

### 1.1 物理原理

气温随海拔升高而降低，根本原因在于气压随高度递减导致空气膨胀绝热冷却。

### 1.2 核心公式

#### 基础公式

```
T(h) = T₀ - Γ × h / 100
```

| 参数 | 含义 | 单位 | 典型值 |
|------|------|------|--------|
| T(h) | 海拔 h 处的年均温 | °C | — |
| T₀ | 海平面（基准面）年均温 | °C | 取决于纬度（见查表） |
| Γ | 气温直减率 | °C/100m | **0.65**（全球平均） |
| h | 海拔 | m | 用户绘制 |

#### 分层直减率（更精确的模型）

| 海拔段 | 直减率 Γ | 适用条件 | 原因 |
|--------|----------|----------|------|
| 0 - 1500m | 0.65 °C/100m | 通用，全球平均 | 自由大气标准直减率 |
| 1500 - 3000m | 0.60 °C/100m | 中海拔 | 水汽含量降低，接近干绝热 |
| 3000 - 5000m | 0.55 °C/100m | 高海拔 | 空气稀薄，辐射冷却增强 |
| > 5000m | 0.50 °C/100m | 极高海拔 | 接近对流层顶 |

### 1.3 季节性修正

直减率随季节变化（夏季对流强，直减率略高）：

| 季节 | Γ修正系数 | 修正后的 Γ |
|------|-----------|------------|
| 夏季 | ×1.10 | 0.72 °C/100m |
| 冬季 | ×0.90 | 0.58 °C/100m |
| 春秋 | ×1.00 | 0.65 °C/100m |

### 1.4 特殊情况规则

| 情况 | 规则 | 机制 |
|------|------|------|
| **逆温层（山谷/盆地冬季）** | IF 地形为封闭盆地 AND 冬季 AND 晴朗无风 THEN T 在谷底反而低于坡面（逆温），ΔT ≈ -0.5°C/100m（冷空气下沉堆积） | 冷空气密度大，沿坡面下沉至谷底 |
| **高原热力效应（夏季）** | IF 平均海拔 > 3000m AND 面积 > 100000 km² AND 夏季 THEN 高原面实际温度比公式计算值高 3-8°C | 高原面接收辐射面积大，感热加热强（青藏高原效应） |
| **冰雪反照率反馈** | IF 海拔 > 雪线 THEN 增加反照率修正：T 额外降低 2-5°C | 冰雪将大部分太阳辐射反射回太空 |

## 二、山脉气象效应

### 2.1 核心概念：山脉如何改变气候

```
                    迎风坡 (Windward)              背风坡 (Leeward)
                    
    盛行风 →  🌬️
                    
                    ☁️☁️☁️ 地形云              🔥 干燥下沉空气
                   /     \                         ↓↓↓
                  /   🌧️🌧️ \                    ☀️☀️☀️
                 /  多雨区   \                 雨影区/焚风
                /             \              (干燥 + 高温)
        海平面 ────────────────────────
                ←—— 上升 ——→              ←—— 下沉 ——→
                   湿绝热冷却                 干绝热增温
                  (Γ ≈ 0.5°C)               (Γ ≈ 1.0°C)
```

### 2.2 输入参数提取

从用户绘制的山脉中提取以下参数：

| 参数 | 符号 | 单位 | 提取方式 |
|------|------|------|----------|
| 山脉高度 | H | m | `max(ridge_elevation) - base_elevation` |
| 山脉宽度 | W | km | 山脚到山脚的水平距离 |
| 山脉走向 | φ | ° | 山脊线的方向角（0=N-S, 90=E-W） |
| 山脉连续性 | C | 0-1 | `1 - (缺口总宽度 / 山脊总长度)` |
| 平均坡度（迎风侧） | θ_w | ° | 迎风侧的平均坡角 |
| 平均坡度（背风侧） | θ_l | ° | 背风侧的平均坡角 |
| 盛行风向 | ψ | ° | 用户设定或由纬度/环流推导 |
| 来流空气水汽含量 | q_in | g/kg | 由上游海洋/水体条件决定 |

### 2.3 山脉-风向夹角效应

山脉对气流的抬升效率取决于**风向与山脉走向的夹角**：

```
抬升效率 η = sin(β)

其中 β = |ψ - φ_perpendicular| 
     φ_perpendicular = (φ + 90°) mod 180°（山脉的法线方向）
```

| 夹角 β | sin(β) | 抬升效率 | 气象效应 |
|--------|---------|----------|----------|
| 90°（风垂直于山脉） | 1.00 | 100% | 最大抬升，最强雨影 |
| 60° | 0.87 | 87% | 强抬升 |
| 45° | 0.71 | 71% | 中等抬升 |
| 30° | 0.50 | 50% | 弱抬升，部分气流绕行 |
| 0°（风平行于山脉） | 0.00 | 0% | 无地形抬升 |

### 2.4 地形抬升与降水增强（迎风坡）

#### 2.4.1 判断是否形成地形降水

```
规则2.1（地形降水触发条件）：
  IF 山脉相对高度 H > 300m
     AND 抬升效率 η > 0.3
     AND 来流空气相对湿度 RH_in > 60%
  THEN 产生地形降水增强效应
  
规则2.2（抬升凝结高度 LCL）：
  LCL = 125 × (T_surface - T_dewpoint)
  单位：m
  IF H > LCL THEN 空气在抬升过程中达到饱和 → 云和降水形成
  IF H < LCL THEN 空气未饱和 → 仅产生地形云（无降水或仅有毛毛雨）
```

#### 2.4.2 迎风坡降水量公式

```
P_windward(h) = P_base × [1 + E × (h/H_optimal) × e^(-(h-H_optimal)² / (2σ²))]

参数：
  P_windward(h)  — 迎风坡海拔 h 处的年降水量 (mm)
  P_base         — 无地形时的基准年降水量 (mm)
  E              — 最大增强因子（取决于来流湿度，1.5 ~ 5.0）
  H_optimal      — 最大降水高度 (m)，通常 2000-3500m（取决于纬度/湿度）
  σ              — 降水垂直分布宽度参数 (m)，通常 800-1500m
```

#### 2.4.3 最大降水高度 H_optimal 查表

| 纬度带 | 气候背景 | H_optimal (m) | 最大增强因子 E_max |
|--------|----------|---------------|-------------------|
| 热带湿润 | 热带海洋气团 | 2000-2500m | 4.0-5.0 |
| 热带干燥 | 副高边缘 | 2500-3000m | 2.0-3.0 |
| 亚热带湿润 | 季风区 | 2000-3000m | 3.0-4.5 |
| 温带海洋性 | 西风带 | 1500-2500m | 3.0-4.0 |
| 温带大陆性 | 内陆 | 2000-3000m | 1.5-2.5 |
| 冷温带 | 高纬度 | 1000-2000m | 1.5-2.0 |

#### 2.4.4 迎风坡降水增强 If-Then 规则

```
规则2.3（迎风坡降水计算）：
  GIVEN 基准降水 P_base (mm)
       山脉高度 H (m)
       来流相对湿度 RH_in (%)
       抬升效率 η (0-1)
  THEN:
    H_optimal = LUT(latitude, climate_background)   // 查上表
    E_max = LUT(latitude, climate_background)        // 查上表
    E = E_max × η × (RH_in / 100)                    // 实际增强因子
    P_windward_max = P_base × (1 + E)
    
    FOR EACH 海拔 h ON 迎风坡:
      IF h <= H_optimal THEN
        P_windward(h) = P_base + (P_windward_max - P_base) × (h / H_optimal)
      ELSE
        P_windward(h) = P_windward_max × e^(-(h - H_optimal)² / 1500²)
```

### 2.5 雨影效应（背风坡降水衰减）

#### 2.5.1 雨影衰减公式

```
P_leeward(d) = P_min + (P_crest - P_min) × e^(-d / D)

参数：
  P_leeward(d)  — 背风坡距山脊 d km 处的年降水量 (mm)
  P_crest       — 山脊处的降水量 (mm) ≈ P_windward(H)
  P_min         — 完全干燥后的最小降水量（可降至 P_base × 0.1）
  d             — 距山脊的水平距离 (km)
  D             — 衰减距离常数 (km)，取决于山脉高度
```

#### 2.5.2 衰减距离 D 查表

| 山脉高度 H (m) | D (km) | 说明 |
|----------------|--------|------|
| 500-1000m | 10-20 km | 低山：雨影效应弱，快速恢复 |
| 1000-2000m | 20-40 km | 中等山脉 |
| 2000-4000m | 40-80 km | 高山：显著雨影 |
| > 4000m | 80-150 km | 极高山：深厚雨影（如安第斯山、喜马拉雅） |

#### 2.5.3 雨影效应 If-Then 规则

```
规则2.4（背风坡降水衰减）：
  GIVEN 山脊降水 P_crest
       基准降水 P_base
       山脉高度 H (m)
       距山脊距离 d (km)
  THEN:
    D = LUT(H)              // 衰减距离查表
    P_min = P_base × 0.10   // 受副高+下沉控制，可降至基准的10%
    
    IF d = 0 THEN
      P_leeward = P_crest
    ELSE
      P_leeward = P_min + (P_crest - P_min) × e^(-d / D)
    
    // 雨影强度分级
    IF P_leeward / P_base < 0.3  → "极强雨影"（如巴塔哥尼亚）
    IF P_leeward / P_base < 0.5  → "强雨影"
    IF P_leeward / P_base < 0.7  → "中等雨影"
    IF P_leeward / P_base >= 0.7 → "弱雨影"
```

### 2.6 焚风效应（背风坡温度升高）

#### 2.6.1 焚风机制

```
空气在迎风坡上升：湿绝热冷却（Γ_moist ≈ 0.5°C/100m）
空气在背风坡下沉：干绝热增温（Γ_dry = 1.0°C/100m）

→ 同一海拔处，背风坡温度 > 迎风坡温度
→ 温度差 ΔT_foehn = (Γ_dry - Γ_moist) × H_condensation / 100
```

#### 2.6.2 焚风升温公式

```
T_leeward(h) = T_windward(h) + ΔT_foehn

ΔT_foehn = (1.0 - 0.5) × (H - LCL) / 100

简化为：
ΔT_foehn ≈ 0.5 × (H - LCL) / 100  (°C)

其中 LCL ≈ 125 × (T_surface - T_dewpoint) （m）
```

#### 2.6.3 焚风强度分级

| ΔT_foehn (°C) | 焚风等级 | 典型山脉高度 |
|---------------|----------|-------------|
| < 2°C | 无显著焚风 | H < 500m 或空气已干燥 |
| 2 - 5°C | 弱焚风 | H 500-1500m |
| 5 - 10°C | 中等焚风 | H 1500-2500m |
| 10 - 20°C | 强焚风 | H 2500-4000m |
| > 20°C | 极强焚风 | H > 4000m（如安第斯山东坡的Zonda风、落基山东坡的Chinook风） |

#### 2.6.4 焚风效应 If-Then 规则

```
规则2.5（焚风温度计算）：
  GIVEN 山脉高度 H (m)
       来流地表温度 T_surface (°C)
       来流露点温度 T_dewpoint (°C)
       背风坡海拔 h_leeward (m)
  THEN:
    LCL = 125 × (T_surface - T_dewpoint)
    
    IF H > LCL AND 抬升效率 η > 0.5 THEN
      // 有焚风效应
      ΔT_foehn = 0.5 × (H - LCL) / 100
      T_leeward(h_leeward) = T_windward(h_leeward) + ΔT_foehn
      
      // 同时相对湿度急剧下降
      RH_leeward = RH_in × e^(-ΔT_foehn / 10)  // 经验公式
      
      // 焚风典型天气
      IF ΔT_foehn > 8 THEN
        weather_type = "typical_foehn"  // 典型焚风：干热风
        可能伴随：气温飙升、湿度骤降、积雪快速融化
    ELSE
      ΔT_foehn = 0  // 无焚风（山脉未达到LCL或风平行于山脉）
```

### 2.7 山脉连续性修正

```
规则2.6（连续性修正）：
  GIVEN 山脉连续性 C (0-1)
       其中 C = 1 - (缺口总宽度 / 山脊线总长度)
  THEN:
    实际抬升效率 η_effective = η × C
    实际雨影强度 = 基准雨影强度 × C²
    
    // 缺口效应
    IF C < 0.5 THEN
      山脉效应显著减弱
      缺口处形成"水汽通道"——迎风坡的湿气可通过缺口到达背风坡
      缺口下游可能形成局地降水增强（"水汽漏"效应）
    
    IF C < 0.3 THEN
      山脉效应基本消失 → 近似于分散的独立山峰
```

### 2.8 山脉效应综合示例

```
输入：
  H = 3000m（山脉高度）
  φ = 90°（东西走向）
  ψ = 180°（南风，即风从南向北吹）→ 风垂直于山脉
  η = sin(90°) = 1.0（最大抬升效率）
  C = 0.95（几乎连续）
  P_base = 800mm
  T_surface = 20°C, T_dewpoint = 14°C
  纬度 = 35°N

计算：
  LCL = 125 × (20 - 14) = 750m
  H_optimal = 2200m（温带海洋性背景）
  E_max = 3.5
  
  // 迎风坡最大降水
  P_windward_max ≈ 800 × (1 + 3.5 × 1.0 × 0.7) = 800 × 3.45 = 2760mm
  
  // 雨影：D ≈ 60km（H=3000m）
  距山脊50km处：P_leeward = 80 + (2760 - 80) × e^(-50/60) 
                            = 80 + 2680 × 0.435
                            = 1246mm（强雨影）
  
  // 焚风
  ΔT_foehn = 0.5 × (3000 - 750) / 100 = 11.25°C（强焚风）
  背风坡1500m处温度比迎风坡同海拔高约11°C
```

## 三、水陆热力差异与季风

### 3.1 大陆度 / 海洋度判定

#### 3.1.1 大陆度指数公式

**Gorczyński 大陆度指数**（最常用）：

```
K = (1.7 × A / sin φ) - 20.4

参数：
  K    — 大陆度指数（无单位）
  A    — 气温年较差（最热月 - 最冷月均温）(°C)
  φ    — 纬度 (°)
  
判定：
  K < 33    → 海洋性气候
  33 ≤ K < 50 → 半海洋性
  50 ≤ K < 65 → 半大陆性
  65 ≤ K < 85 → 大陆性
  K ≥ 85    → 极端大陆性
```

#### 3.1.2 距海距离与大陆度 If-Then 规则

如果用户尚未运行气候模拟，可通过距海距离**估算**大陆度：

```
规则3.1（距海距离估算大陆度）：
  GIVEN 距最近海洋距离 d_ocean (km)
       盛行风向 ψ
       该地在 ψ 方向上距海岸的距离 d_windward_ocean (km)
       
  THEN:
    // 计算"有效海洋距离"（考虑盛行风向）
    IF 该地位于盛行风向的向岸侧 AND d_ocean < 500km THEN
      d_effective = d_windward_ocean  // 海洋影响沿风向上游
    ELSE
      d_effective = d_ocean  // 使用欧几里得距离
    
    // 大陆度估算
    K_estimated = 10 + 40 × (1 - e^(-d_effective / 500))
    
    // 年温差估算
    A_estimated = 8 + 30 × (1 - e^(-d_effective / 800))
```

#### 3.1.3 距离-大陆度分级

| 距海距离 (km) | 大陆度 K | 年温差 (°C) | 气候类型 | 典型地区 |
|---------------|----------|-------------|----------|----------|
| < 50 | 10-25 | 8-15 | 强海洋性 | 沿海岛屿、半岛尖端 |
| 50-200 | 25-40 | 15-22 | 海洋性 | 西欧沿海、日本太平洋侧 |
| 200-500 | 40-55 | 22-30 | 半大陆性 | 东欧、中国华北沿海 |
| 500-1000 | 55-70 | 30-38 | 大陆性 | 西西伯利亚、北美大平原 |
| 1000-2000 | 70-85 | 38-50 | 强大陆性 | 东西伯利亚、中亚 |
| > 2000 | > 85 | > 50 | 极端大陆性 | 西伯利亚腹地（维尔霍扬斯克） |

#### 3.1.4 If-Then 规则

```
规则3.2（海陆热力调节）：
  GIVEN 距海距离 d (km)
       （可选）盛行风向信息
  THEN:
    // 冬季升温（海洋释放夏季储存的热量）
    ΔT_winter = +15 × e^(-d / 300)   (°C, 相对于同纬度大陆内部)
    
    // 夏季降温（海洋吸收热量）
    ΔT_summer = -8 × e^(-d / 300)    (°C, 相对于同纬度大陆内部)
    
    // 年温差缩小
    A = A_continental - 23 × e^(-d / 300)
    
    // 湿度增加
    ΔRH = +25 × e^(-d / 500)         (%, 相对于同纬度大陆内部)

规则3.3（水体面积修正）：
  IF 最近水体为湖泊（非海洋）THEN
    // 湖泊调节能力弱于海洋
    等效距离 d_effective = d × (1 + 500/sqrt(A_lake))
    其中 A_lake 为湖泊面积 (km²)
    （小湖泊等效距离远大于实际距离）
```

### 3.2 季风环流强度判定

#### 3.2.1 季风形成的必要条件

季风需要三个条件同时满足：

```
1. 巨大的海陆热力差异（大陆面积 + 相邻海洋面积要足够大）
2. 合适的纬度范围（5°-35° 最佳，最远可到50°）
3. 地形不封锁气流通道
```

#### 3.2.2 季风强度指数公式

```
M = α × ΔT_summer × (1 + β × L_land) × (1 + γ × L_ocean) × (1 - δ × R_terrain)

参数：
  M             — 季风强度指数 (0-100)
  ΔT_summer     — 夏季大陆与相邻海洋的表层温度差 (°C)
  L_land        — 大陆在盛行风向上的纵深 (1000km 为单位)
  L_ocean       — 相邻海洋在盛行风向上的纵深 (1000km 为单位)
  R_terrain     — 地形阻力系数 (0-1, 0=完全开放, 1=完全封锁)
  α,β,γ,δ      — 经验系数
```

**简化经验公式**（适用于工具实现）：

```
M = 15 × ΔT_summer × (1 + 0.3 × D_land) × (1 + 0.2 × D_ocean) × (1 - 0.8 × R_terrain)

其中 D_land = 大陆纵深(km) / 1000
     D_ocean = 海洋纵深(km) / 1000
     R_terrain = 地形阻力 (0-1, 见下文)
```

#### 3.2.3 季风强度分级

| M 值范围 | 季风强度 | 降水集中度 | 典型地区 |
|----------|----------|-----------|----------|
| M ≥ 80 | 极强季风 | 夏季降水 > 80% | 南亚（印度西海岸）、东南亚 |
| 60 - 79 | 强季风 | 夏季降水 65-80% | 东亚（中国东部、日本）、西非 |
| 40 - 59 | 中等季风 | 夏季降水 50-65% | 澳大利亚北部、东非 |
| 20 - 39 | 弱季风 | 夏季稍多但不显著 | 美国东南部（副高西缘型） |
| M < 20 | 非季风区 | 降水均匀分配 | 西欧、赤道雨林 |

#### 3.2.4 地形对季风的阻挡

```
规则3.4（地形阻力计算）：
  GIVEN 沿海山脉的高度 H_coastal (m)
       山脉离海岸的距离 d_coastal (km)
       山脉连续性 C (0-1)
       是否有大型河口/平原缺口
  THEN:
    // 沿海山脉的阻挡效率
    IF d_coastal < 100km AND H_coastal > 1000m THEN
      R_terrain = min(1.0, C × (H_coastal / 3000) × (1 - d_coastal / 200))
      // 山脉越高、离海岸越近、越连续 → 阻挡越强
    ELSE
      R_terrain = 0  // 无显著阻挡
    
    // 缺口效应
    IF 存在大型河口或平原缺口（宽度 > 20km）THEN
      R_terrain = R_terrain × (1 - gap_width / mountain_total_width)
      // 缺口允许季风穿透（如恒河平原缺口使季风深入印度北部）

规则3.5（季风深入内陆的距离）：
  GIVEN 季风强度 M
       地形阻力 R_terrain
  THEN:
    // 季风有效影响距离
    D_monsoon_penetration = M × (1 - R_terrain) × 20  (km)
    
    IF D_monsoon_penetration > 3000 THEN
      "季风可深入内陆超过3000km"（如南亚季风可达青藏高原南麓）
    
    // 降水沿程衰减
    P(x) = P_coast × e^(-x / D_monsoon_penetration)
    其中 x = 距海岸距离 (km)

规则3.6（海陆热力差异的计算）：
  GIVEN 用户绘制的陆地范围
       相邻海洋（或由用户指定海洋位置）
  THEN:
    ΔT_summer ≈ 大陆7月均温 - 相邻海洋表层7月均温
    可用以下查表估算：
    
    大陆面积 (km²)    ΔT_summer 估算 (°C)
    < 100万             2-5°C
    100万 - 500万        5-12°C
    500万 - 2000万      12-20°C
    > 2000万 (欧亚级别)  15-25°C
    
    // 更大面积的大陆 → 更强的夏季加热 → 更大的ΔT → 更强的季风
```

### 3.3 综合判定：该地是否有季风？

```
规则3.7（季风存在性判定）：
  GIVEN 纬度 φ
       大陆面积 A_land（以及是否在大陆东岸）
       相邻海洋面积 A_ocean
       海陆夏季温差 ΔT_summer
       地形阻力 R_terrain
  THEN:
    IF φ 在 5°-35° 范围内
       AND A_land > 200万 km²
       AND A_ocean > 500万 km²  // 相邻暖洋足够大
       AND ΔT_summer > 8°C
       AND R_terrain < 0.7
    THEN:
      该地为季风区
      计算 M
      IF M >= 40 THEN 显著季风
      IF M >= 60 THEN 强季风
    ELSE IF φ 在 35°-50° 范围内
       AND A_land > 500万 km²
       AND ΔT_summer > 12°C
    THEN:
      温带季风区（Dwa/Dwb型）
    ELSE:
      非季风区 → 降水的季节性由其他机制决定（ITCZ移动、西风气旋等）
```

## 四、水体调节作用

### 4.1 大型湖泊/内陆海的微气候调节

#### 4.1.1 水体调节的核心物理参数

| 参数 | 符号 | 典型值（湖泊） | 典型值（海洋） | 说明 |
|------|------|---------------|---------------|------|
| 比热容 | c | 4.2 J/g·°C | 4.2 J/g·°C | 约为陆地(0.8-1.5)的3-5倍 |
| 热传导深度 | z | 1-10m（浅湖）/ 20-50m（深湖） | 50-200m（混合层） | 决定热量储存量 |
| 蒸发率 | E | 500-1500 mm/yr | 1000-2000 mm/yr | 取决于温度/风速 |
| 反照率 | α | 0.06-0.10 | 0.05-0.10 | 远低于陆地(0.15-0.35) |

#### 4.1.2 水体温度调节范围公式

```
规则4.1（水体温度调节）：
  GIVEN 水体面积 A_water (km²)
       水体平均深度 Z_water (m)
       观测点距水体边缘距离 d (km)
       观测点在盛行风向的下风向（或上风向）
  THEN:
  
  // 1. 水体热容量
  C_water = A_water × Z_water × c_water  (相对值)
  C_water_norm = min(1.0, C_water / C_reference)
  其中 C_reference = 10000 × 20 × 4.2（10000km² × 20m深的湖泊作为参考）
  
  // 2. 影响范围（下风向）
  D_influence = 5 × sqrt(A_water) × (Z_water / 10)^0.3  (km)
  // 大水体影响范围可达50-200km
  
  // 3. 温度调节量
  ΔT_winter_warming = 12 × C_water_norm × e^(-d / D_influence) × WF  (°C)
  ΔT_summer_cooling = -6 × C_water_norm × e^(-d / D_influence) × WF  (°C)
  
  其中 WF = wind_factor:
    IF 观测点在盛行风向下游（风从水面吹向该点）AND d < D_influence THEN
      WF = cos(θ)  // θ = 风向与该点到水体连线方向的夹角
    ELSE IF 观测点在上游 THEN
      WF = 0.15  // 仅通过辐射和弱对流产生微弱影响
```

#### 4.1.3 水体分级与影响参数

| 水体类型 | 面积 A (km²) | W_inter 参考深度 | D_influence (km) | ΔT_winter max | ΔT_summer max |
|----------|-------------|------------------|------------------|---------------|---------------|
| 小湖 | 1-10 | 5m | 0.5-2 | +1-2°C | -0.5-1°C |
| 中湖 | 10-100 | 10m | 2-8 | +2-4°C | -1-2°C |
| 大湖 | 100-1000 | 15m | 8-30 | +4-7°C | -2-4°C |
| 巨型湖 | 1000-10000 | 20m | 30-80 | +7-10°C | -4-6°C |
| 内陆海 | 10000-100000 | 50m | 80-200 | +10-13°C | -6-8°C |
| 真正海洋 | > 100000 | 100m+ | 100-400 | +13-15°C | -7-9°C |

#### 4.1.4 湿度增加效应

```
规则4.2（水体湿度增强）：
  GIVEN 水体面积 A_water
       观测点距水体距离 d (km)
       水体温度 T_water (°C)
       环境温度 T_air (°C)
       风速  v_wind (m/s)
  THEN:
  
  // 蒸发潜力（基于 Dalton 蒸发公式简化）
  E_pot = (e_sat(T_water) - e_actual) × v_wind^0.5
  
  // 下风向湿度增加
  ΔRH = 30 × C_water_norm × e^(-d / D_humidity) × WF  (%)
  其中 D_humidity = 3 × sqrt(A_water) (km)  // 湿度影响范围通常小于温度
  
  // 当冷水体（T_water < T_air）时 → 抑制蒸发 → 可能形成雾
  IF T_water < T_air AND RH_ambient > 80% THEN
    fog_probability = 0.7
    fog_type = "advection_fog"  // 平流雾（如旧金山金门、秘鲁沿岸）
```

#### 4.1.5 湖效应降水（Lake-Effect Snow/Rain）

```
规则4.3（湖效应降水触发）：
  GIVEN 水体面积 A_water (>500km² 才可能产生显著湖效应)
       水体温度 T_water
       来流空气温度 T_air
       来流空气湿度 RH_air
       风在水面上的路径长度（吹程，fetch） L_fetch (km)
       风吹过水面后的登陆地形（是否有迎风坡）

  THEN:
    // 湖效应必要条件
    ΔT_water_air = T_water - T_air
    
    IF ΔT_water_air > 13°C
       AND L_fetch > 50km
       AND A_water > 500km²
       AND RH_air < 80%  // 干燥冷空气更有利于湖效应
    THEN:
      湖效应降水触发！
      
      // 降水强度估算
      P_lake_effect = 5 × (ΔT_water_air - 10) × (L_fetch / 50)^0.5 × 地形增强因子
      
      地形增强因子:
        IF 登陆点有迎风坡(坡度 > 3%) THEN
          地形增强因子 = 1.5 ~ 3.0
        ELSE
          地形增强因子 = 1.0
      
      // 典型湖效应降雪量
      IF T_air < 0°C THEN
        降雪量(cm) ≈ P_lake_effect × 0.7  // 经验雪水比
      ELSE
        降雨量(mm) ≈ P_lake_effect × 0.5

    湖效应降水强度参考：
      大湖(如北美五大湖)：年湖效应雪可达200-500cm
      日本海(冬季西北季风越过暖海面)：日本北陆沿岸年降雪可达800-1500cm
      
规则4.4（湖风/海风环流）：
  GIVEN 水体-陆地温差 ΔT_land_water
       （白天：陆地热；夜间：陆地冷）
  THEN:
    // 白天湖风（水面 → 陆地）
    IF 白天 AND ΔT_land_water > 5°C THEN
      湖风/海风风速 ≈ 3 + (ΔT_land_water - 5) × 0.8  (m/s)
      湖风深入内陆距离 ≈ 风速 × 10  (km)
      // 通常深入内陆10-30km
      
    // 夜间陆风（陆地 → 水面）
    IF 夜间 AND ΔT_water_land > 3°C THEN
      陆风风速 ≈ 2 + (ΔT_water_land - 3) × 0.5  (m/s)
      陆风深入水面距离 ≈ 风速 × 3  (km)
      // 通常仅深入水面5-15km
```

## 五、综合规则引擎：地形 → 气候类型再判定

### 5.1 地形修正后的柯本类型重新计算

```
当用户绘制地形后，按以下顺序依次修正基准气候参数：

1. 海拔修正温度
   T' = T - Γ × h / 100
   
2. 距海距离修正大陆度
   A' = A + 30 × (1 - e^(-d/800))
   (距海越远，年温差越大)
   
3. 山脉修正降水
   IF 位于迎风坡 → P' = P × (1 + E_orographic)
   IF 位于背风坡 → P' = P_min + (P_crest - P_min) × e^(-d/D)
   
4. 水体修正温度/湿度
   IF 位于大型水体下风向 AND d < D_influence →
     T'_winter = T_winter + ΔT_winter_warming
     T'_summer = T_summer + ΔT_summer_cooling
     RH' = RH + ΔRH

5. 用修正后的参数重新判定柯本类型
```
