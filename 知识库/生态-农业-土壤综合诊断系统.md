# 生态-农业-土壤综合诊断报告系统

## 基于 Holdridge + FAO AEZ + WRB 的三层推理引擎

---

## 系统总览

```
输入参数（来自气候引擎 + 用户地形图）
    │
    ├── 年均温 T (°C)、年降水 P (mm)
    ├── 最冷月/最热月均温、年温差 ΔT
    ├── 海拔 h (m)、坡度 θ (°)、坡向
    ├── 母质类型（花岗岩/玄武岩/石灰岩/砂岩/冲积物…）
    ├── 地下水位深度、排水条件
    └── 柯本气候类型（来自气候判定矩阵）
         │
         ├──→ 第一层 [Holdridge] → 潜在自然植被类型 + 生命带
         ├──→ 第二层 [FAO AEZ]   → 农作物适宜性推荐
         └──→ 第三层 [WRB]       → 推断土壤类型
              │
              └──→ 综合诊断报告
```

## 一、Holdridge 生命带系统 → 潜在自然植被推断

### 1.1 核心参数计算

Holdridge 系统使用三个核心变量：

```
(1) 生物温度 BT（Biotemperature）

    BT = Σ(T_i 的贡献) / 12

    其中 T_i 为月均温，贡献规则：
      IF T_i < 0°C  THEN 贡献 = 0（冻结，生物活动停止）
      IF T_i > 30°C THEN 贡献 = 30（高温对植物生长不再额外促进）
      ELSE 贡献 = T_i

    BT 的单位：°C
    BT 的含义：植物生长的有效热量


(2) 年降水量 P (mm)

(3) 潜在蒸散率 PER（Potential Evapotranspiration Ratio）

    PET_raw = 58.93 × BT  (mm/yr, Holdridge 原始经验公式)

    【关键修正——太阳辐射纬度校正】
    Holdridge 原始公式 PET=58.93×BT 隐含了热带太阳辐射强度的前提。
    该系数 58.93 是从低纬度数据拟合得出的——假设每°C生物温度对应
    约 59mm 的蒸发潜力。但蒸发潜力本质上取决于"可用于驱动蒸发的能量"
    ——即太阳辐射——而非仅仅温度。

    同一 BT 在不同纬度意味着完全不同的蒸发潜力：
      - BT=6°C 在赤道 → 不存在(赤道BT>24)
      - BT=6°C 在 59°N → 太阳辐射仅为赤道的 ~51%
    如果不做辐射校正，高纬度的 PET 会被系统性高估 → PER 虚高
    → 森林被误判为草原。这不是 Holdridge 系统的问题，而是 58.93 这个
    常数被当作"普适常数"使用时的尺度错误。

    【修正公式】
      PET = 58.93 × BT × f_rad(|φ|)

      f_rad(φ) = I_annual(φ) / I_annual(0°)
        = I_annual(φ) / 300

      其中 I_annual(φ) 为纬度 φ 处的年均太阳辐射(W/m²), 查表:

      | 纬度 | 0° | 15° | 30° | 45° | 60° | 75° | 90° |
      |------|-----|------|------|------|------|------|------|
      | I(W/m²)| 300 | 290 | 260 | 212 | 150 | 78 | 0 |
      | f_rad | 1.00 | 0.97 | 0.87 | 0.71 | 0.50 | 0.26 | 0.00 |

      中间纬度线性插值。

      物理含义: f_rad 代表了"该纬度可用于驱动蒸发的太阳能量占赤道的比例"。
      赤道 f_rad=1.0 → PET 不变 → 与原始 Holdridge 一致。
      纬度越高 f_rad 越小 → PET 越低 → 同样 BT 和 P 下 PER 越低 → 越湿润。

    PER = PET / P
```

### 1.2 纬度带（Latitudinal Belt）判定

| 纬度带 | BT 范围 | 植被形态特征 | 对应的柯本类型 |
|--------|---------|-------------|---------------|
| 极地 (Polar) | BT < 1.5°C | 无乔木，仅有地衣/苔藓 | EF |
| 亚极地 (Subpolar) | 1.5 ≤ BT < 3°C | 矮灌丛、湿苔原 | ET |
| 北方带/寒温带 (Boreal) | 3 ≤ BT < 6°C | 针叶林 | Dfc/Dfd/Dwc |
| 冷温带 (Cool Temperate) | 6 ≤ BT < 12°C | 针阔混交林、落叶阔叶林 | Dfb/Dwb/Cfb |
| 暖温带 (Warm Temperate) | 12 ≤ BT < 18°C | 常绿阔叶林、落叶林 | Cfa/Cfb/Csa |
| 亚热带 (Subtropical) | 18 ≤ BT < 24°C | 亚热带常绿林、硬叶林 | Cfa/Cwa/Csa |
| 热带 (Tropical) | BT ≥ 24°C | 热带雨林、季雨林、稀树草原 | Af/Am/Aw |

### 1.3 湿度省（Humidity Province）判定

| 湿度省 | PER 范围 | 年降水参考 (BT=24时) | 植被水分状态 |
|--------|----------|----------------------|-------------|
| 超湿润 (Superhumid) | PER < 0.125 | P > 11300mm | 水分严重过剩，淋溶极强 |
| 过湿润 (Perhumid) | 0.125 ≤ PER < 0.25 | 5650-11300mm | 水分过剩，雨林 |
| 湿润 (Humid) | 0.25 ≤ PER < 0.50 | 2830-5650mm | 水分充足，森林 |
| 亚湿润 (Subhumid) | 0.50 ≤ PER < 1.00 | 1415-2830mm | 水分基本充足，林地或高草 |
| 半干旱 (Semiarid) | 1.00 ≤ PER < 2.00 | 710-1415mm | 水分不足，草原/稀树草原 |
| 干旱 (Arid) | 2.00 ≤ PER < 4.00 | 355-710mm | 严重缺水，荒漠灌丛 |
| 极干旱 (Perarid) | PER ≥ 4.00 | P < 355mm | 极端缺水，荒漠/无植被 |

### 1.4 生命带 - 潜在自然植被推断矩阵

```
规则1.1（Holdridge生命带判定）：
  GIVEN 用户绘制的区域 → 气候模拟输出：
    年均温 T (°C)、各月均温 T_i (°C)
    年降水 P (mm)
  THEN:
    BT = Σ(max(0, min(30, T_i))) / 12
    PET = 58.93 × BT
    PER = PET / P
    
    纬度带 = LUT(BT)      // 查 1.2 表
    湿度省 = LUT(PER)      // 查 1.3 表
    生命带 = 纬度带 × 湿度省 → 查 1.5 表
    潜在植被 = 生命带 → 对应植被类型
```

### 2.2 作物水分需求分类

| 水分需求等级 | 生长季降水需求 | 典型作物 |
|-------------|--------------|----------|
| 高需水 (hydrophytic) | >1000mm 或灌溉 | 水稻、甘蔗 |
| 中高需水 | 600-1000mm | 玉米、大豆、棉花、马铃薯 |
| 中等需水 | 400-600mm | 小麦、大麦、油菜、甜菜、高粱 |
| 低需水 (xerophytic) | 250-400mm | 粟/小米、燕麦、黑麦 |
| 极低需水 (干旱作物) | <250mm (需灌溉) | 仅少数耐旱作物（高粱、粟）可旱作 |

### 2.3 土壤质地适配

| 土壤质地 | 排水性 | 持水性 | 适种作物 |
|----------|--------|--------|---------|
| 砂土 (Sand) | 极快 | 差 | 花生、马铃薯、胡萝卜、芦笋 |
| 砂壤土 (Sandy Loam) | 快 | 较差 | 马铃薯、花生、烟草、大麦 |
| 壤土 (Loam) | 适中 | 好 | **最适耕作土壤** — 几乎所有作物 |
| 粉砂壤土 (Silt Loam) | 较慢 | 很好 | 小麦、玉米、大豆、甜菜 |
| 黏壤土 (Clay Loam) | 慢 | 很好 | 小麦、大麦、油菜、牧草 |
| 黏土 (Clay) | 极慢 | 极好 | 水稻（水田）、牧草；旱作需排水 |

## 三、WRB 土壤分类 → 土壤类型推断

### 3.1 土壤诊断逻辑（气候 · 地形 · 母质 · 植被 → 土壤）

WRB 使用诊断层和诊断特性来定义土壤类型。在用户工具中，我们通过四个可观测/可计算的变量来反推土壤类型。

```
土壤类型 = f(气候, 地形, 母质, 植被)

输入：
  气候：温度带 + 降水/水分状况 → 决定风化强度和方向
  地形：海拔 + 坡度 + 排水条件 → 决定侵蚀/堆积/潜育化
  母质：岩石类型 → 决定矿物组成和风化产物
  植被：Holdridge生命带 → 决定有机质输入和生物扰动
```

### 3.2 气候驱动的成土过程

| 气候条件 | 主导成土过程 | 诊断层/特性 | 代表性 WRB 土壤 |
|----------|------------|-----------|---------------|
| 炎热+极湿 (Af/Am) | 强化学风化、脱硅富铝化、盐基淋失 | ferralic (铁铝层) | **Ferralsols** (砖红壤/铁铝土) |
| 炎热+季节性湿 (Aw) | 中度风化、黏粒形成与迁移、聚铁网纹化 | argic (黏化层), plinthic | **Acrisols/Lixisols/Nitisols** |
| 温暖+湿润 (Cfa/Cfb) | 中度风化、黏粒迁移、盐基中度淋失 | argic (黏化层), cambic (雏形层) | **Luvisols/Alisols/Cambisols** |
| 温带草原 (BSk/Dfa 较干边缘) | 有机质深厚积累、钙积化 | mollic (暗沃层), calcic (钙积层) | **Chernozems** (黑钙土), **Kastanozems** (栗钙土), **Phaeozems** (暗色土) |
| 冷凉+湿润 (Dfc/Dwc, 针叶林) | 强酸性淋溶、灰化(铁铝螯合淋移) | spodic (灰化层) | **Podzols** (灰化土) |
| 干旱 (BWh/BWk) | 钙积化、石膏化、盐渍化 | calcic, gypsic, salic | **Calcisols**, **Gypsisols**, **Solonchaks** |
| 半干旱 (BSh/BSk) | 钠质化、黏粒迁移 | natric (碱化层) | **Solonetz** (碱土) |
| 永久冻土 (ET/Dfd) | 冻融扰动、冰裂隙 | cryic (冰冻特性) | **Cryosols** (冻土) |
| 积水/高地下水位 | 潜育化（铁还原） | gleyic (潜育特性) | **Gleysols** (潜育土), **Stagnosols** (滞水土) |
| 有机质积累(涝) | 泥炭化 | histic (有机表层) | **Histosols** (有机土) |

### 3.4 地形-土壤关系

| 地形位置 | 海拔 | 坡度 | 排水条件 | 主导过程 | 典型土壤 |
|----------|------|------|---------|---------|---------|
| 陡峭山脊/崖壁 | 不限 | >30° | 过快 | 侵蚀 >> 成土 | **Leptosols** (薄层土/石质土) |
| 山坡中上部 | 不限 | 15-30° | 快 | 侵蚀 + 弱成土 | **Regosols**, **Cambisols** (雏形土) |
| 山坡下部/麓 | 不限 | 5-15° | 适中 | 堆积 + 成土 | **Cambisols**, **Luvisols** |
| 平坦台地/高原 | 不限 | <3° | 适中 | 稳定成土 | 取决于气候（最深发育） |
| 河谷/冲积平原 | 低 | <2° | 适中-不良 | 冲积 + 潜育 | **Fluvisols**, **Gleysols** |
| 盆地/洼地 | 低 | <2° | 不良-极差 | 积水 + 潜育/有机积累 | **Gleysols**, **Histosols**, **Vertisols** |
| 高海拔 (>3000m) | 高 | 不限 | 取决于坡度 | 冻融 + 弱风化 | **Cryosols**, **Leptosols** |

### 3.5 植被-土壤的有机质输入

| 植被类型 (Holdridge) | 有机质输入特征 | 对土壤的影响 | 相关诊断层 |
|---------------------|--------------|-------------|-----------|
| 热带雨林 | 大量凋落物但快速分解 | 养分在生物量中循环，土壤本身贫瘠 | ferralic |
| 热带稀树草原 | 草类根系深厚，火灾频繁 | 有机质来自根系，表层可能有暗色层 | mollic (弱) |
| 温带落叶林 | 年度落叶，中等分解速率 | 形成 moder/mull 腐殖质 | mollic/cambic |
| 温带草原 | 海量细根系周转，蚯蚓活跃 | 深厚暗沃层 (黑色、团粒结构) | **mollic** ← Chernozem 的标志 |
| 针叶林(泰加) | 针叶凋落，酸性、分解极慢 | 形成 mor 粗腐殖质 → 强酸淋溶 | spodic |
| 苔原 | 极少有机质输入，分解极慢 | 泥炭积累(涝时) / 裸矿质土(干时) | cryic, histic |

### 3.6 WRB 土壤类型综合推断矩阵

```
规则3.1（土壤类型推断）：
  GIVEN 柯本气候类型（或 Holdridge 生命带）
       地形位置（海拔 + 坡度 + 排水条件）
       母质类型（用户选择）
       潜在植被类型（来自 Holdridge 推断）
  THEN:
    主要成土过程 = 诊断逻辑矩阵（气候 + 地形 + 母质 + 植被）→ 综合
    推断土壤 = 主要成土过程 + 诊断层 → WRB RSG
    
    置信度:
      "高(>80%)"：所有四个因素指向同一土壤类型
      "中(50-80%)"：2-3个因素一致
      "低(<50%)"：因素矛盾，给出多个可能
```

### 3.7 推断逻辑（If-Then 判定链）

```
规则3.2（按优先级逐一排查）：
  
  1. 地形优先检查（地形可推翻一切气候推断）：
    IF 坡度 > 30°                            → Leptosols  (薄层土)
    IF 位于河漫滩/三角洲/近代冲积平原         → Fluvisols   (冲积土)
    IF 母质 = 火山灰/火山碎屑                  → Andosols    (火山灰土)
    IF 长期积水 OR 地下水位 < 0.5m             → Gleysols    (潜育土)
    IF 有机质积累 > 40cm (泥炭/沼泽)            → Histosols   (有机土)
    IF 永久冻土 (年均温 < −2°C)                → Cryosols    (冻土)
  
  2. 气候-植被综合检查：
    IF 热带 + 湿润(Af/Am) + 平坦/台地           → Ferralsols  (铁铝土)
    IF 热带 + 干湿季(Aw) + 黏质母质              → Vertisols   (变性土)
    IF 温带草原(BSk) + 平坦 + 高草               → Chernozems  (黑钙土)
    IF 温带草原(BSk) + 平坦 + 短草               → Kastanozems (栗钙土)
    IF 温带草原(Dfa较干) + 湿润                  → Phaeozems   (暗色土)
    IF 冷温带/北方 + 针叶林 + 酸性母质           → Podzols     (灰化土)
    IF 沙漠(BWh/BWk) + 钙质母质或长期干旱        → Calcisols   (钙积土)
    IF 沙漠 + 蒸发强烈 + 排水不良                → Solonchaks  (盐土)
    IF 半干旱(BSh/BSk) + 钠质化                  → Solonetz    (碱土)
  
  3. 风化程度检查（温度+降水驱动的强度）：
    IF 年均温 > 22°C AND 年降水 > 1500mm        → 强风化(ferralic)
    IF 年均温 15-22°C AND 年降水 800-1500mm     → 中度风化( argic )
    IF 年均温 5-15°C AND 年降水 400-800mm       → 弱风化(cambic)
    IF 年均温 < 5°C OR 年降水 < 400mm           → 极弱风化
  
  4. 默认/过渡类型：
    IF 不满足上述任何特殊条件                   → Cambisols   (雏形土)
```
