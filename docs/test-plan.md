# Test Plan: TerraDiagnosis / 地脉·镜

Generated by /plan-eng-review on 2026-05-23
Branch: main

## Affected Pages
- `/` — 3D terrain sandbox (SPA): edit view + simulation view + planet archive panel

## Key Interactions to Verify
- Brush sculpt (raise/lower/flatten/smooth) — contour lines update in real time
- Terrain preset selection — heightmap instantly replaced
- Lock button: edit → simulating transition with loading UI
- Post-lock: rivers animate, climate zones color terrain, water animates
- Planet archive: terrain classification, climate zones, hydrology summary
- Share URL: encode → copy → open in new tab → identical terrain renders
- Undo/redo: sculpt → undo restores → redo re-applies
- Web Worker recovery: kill worker → error shown → retry succeeds

## Edge Cases
- Double-click lock button prevented by mode guard
- Completely flat terrain → "该星球无天然河流" message
- Enclosed basin → lake forms at lowest point
- Invalid share URL → graceful fallback to default terrain
- WebGL context lost → recovery prompt + auto-retry
- Worker timeout → degraded mode (hydrology only)

## Critical Paths
- **Demo script**: Preset → Sculpt → Lock → Rivers + Climate → Archive → Share (60 sec)
- **Share round-trip**: encode → decode must produce identical heightmap
- **Brush math**: Gaussian falloff correctness

## Unit Test Specs

### brush.test.ts (Vitest)
- raise applies gaussian falloff correctly
- lower applies negative gaussian correctly
- flatten converges to local average
- smooth reduces local variance
- brush edge does not affect distant cells

### d8.test.ts (Vitest)
- flow direction on uniform slope → all point downhill
- flow direction on flat terrain → no direction
- flow accumulation sums upstream cells
- lake detected at local minimum below threshold

### share.test.ts (Vitest)
- encode(decode(x)) === x for random heightmaps
- decode rejects hash longer than MAX_LENGTH
- decode rejects hash that decompresses to > MAX_SIZE
- decode rejects malformed base64

### demo-script.md (Manual)
60-second booth script with emotional arc annotations (see design doc).
