# TerraDiagnosis / 地脉·镜

> 地理世界观 OC 构建工具 —— 推沙成世界，一镜诊山河。

## 项目简介

一个面向文娱创作者的交互式星球地理生成与诊断平台。浏览器打开即可使用。

**创作**：推拉地形，在国家地理风格的卫星图上雕刻大陆、山脉、海岸线。

**诊断**：锁定地形后，系统运行完整的地理模拟——水文、气候、土壤、植被、矿产——为你创造的世界赋予科学依据。

**构建**：放置图钉标记城邦与秘境，圈定王国与行省边界，生成属于你的世界观设定集。

## 核心特色

- **全星球尺度**：512×512 栅格覆盖 360°×180° 经纬度，每一格都有纬度、经度、海拔、坡度、坡向
- **科学地理模拟**：Köppen-Geiger 气候分类 + Holdridge 生命带 + WRB 土壤分类 + 原生植被 + 野生动物 + 矿产分布 + 城镇发展潜力
- **专业制图渲染**：国家地理风格分层设色（6 级地形 + 3 级海洋 + 5 级植被），等高线分层设色图模式
- **文娱创作工具**：自定义图钉（标记地点 + 文字描述）、自定义选区（圈定疆域 + 区域命名）
- **知识引擎**：14 个地理知识模块驱动模拟，涵盖气候、土壤、植物、动物、矿产、城镇

## 技术栈

Three.js + React Three Fiber · 自定义 GLSL 着色器 · Zustand 状态管理 · Web Worker 模拟 · Vite 构建 · GitHub Pages 部署

## 快速开始

```bash
npm install
npm run dev
```

## 操作

| 操作 | 方式 |
|------|------|
| 左键拖拽 | 绘制地形 |
| 右键拖拽 | 平移画面 |
| 滚轮 | 缩放（拉远自动切换俯瞰 + 等高线） |
| Ctrl+Z/Y | 撤销/重做 |
| 放置图钉 | 模拟完成后点击地形放置，输入内容 |

## 知识库

`知识库/` 目录包含 14 个地理知识模块：气候形成框架、地形-气候转化规则、柯本气候分类矩阵、气候-土壤规则库、气候-原产植物库、气候-野生动物属库、生态-农业-土壤诊断系统、矿产分布规则库、城镇发展潜力评估引擎、基础地理参数推断引擎、水文推断引擎等。
