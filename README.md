# TerraDiagnosis / 地脉·镜

> 3D 交互式星球地形沙盘 —— 推沙成世界，一镜诊山河。

## 项目简介

灵感来自科技馆的 AR 实体沙盘教具。纯数字版，浏览器打开即可使用。画布代表整颗星球（360°×180° 经纬度）。

**创作阶段**：推拉地形，实时卫星图级渲染。隆起、削低、平滑、冰川侵蚀四种笔刷。

**揭示阶段**：锁定地形后，系统接管——水文模拟（D8 河流追踪）、气候模拟（纬度驱动+地形雨+雨影+大陆度）、Köppen-Geiger 气候分类、Holdridge 生命带判定、WRB 土壤推断、原生植被与农作物推荐。

**图钉系统**：放置图钉，模拟后悬停查看该点的经纬度、海拔、坡度坡向、气压带、气候类型、动植物、土壤矿产、城镇发展潜力。

## 技术栈

| 层 | 技术 |
|---|---|
| 3D 渲染 | Three.js + React Three Fiber |
| 地形着色 | 自定义 GLSL ShaderMaterial（卫星图级渲染） |
| 水体着色 | 动画 ShaderMaterial（河流流动+湖泊波纹） |
| 状态管理 | Zustand |
| 水文模拟 | D8 流向 + 洼地填充 + 河道路径追踪 (Web Worker) |
| 气候模拟 | 纬度驱动 + 地形雨 + 雨影 + BFS 海岸距离 + 大陆度 |
| 气候分类 | Köppen-Geiger + Holdridge 生命带 |
| 土壤推断 | WRB 国际土壤分类 |
| 知识库 | 6 模块地理知识引擎（气候/土壤/植物/动物/矿产/城镇） |
| 构建 | Vite |
| 部署 | GitHub Pages (纯静态) |

## 架构

```
主线程 (UI)
├── React UI (笔刷/图钉/锁定/预设)
├── R3F 3D 场景渲染
│   ├── TerrainMaterial (编辑模式 — 卫星图着色)
│   │   └── 纬度驱动雪线 + 极地冰盖 + 气候纹理叠加
│   ├── WaterMaterial (河流流动 + 湖泊波纹动画)
│   └── 河流线 (CatmullRom 平滑 D8 路径)
├── 星球档案面板 (地形/气候/水文/土壤/植物)
├── 图钉卡片 (11 项地方诊断)
└── Zustand Store { heightmap, mode, brush, markers, riverData }
            │ postMessage + transfer
            ▼
Web Worker (模拟引擎)
├── 水文模型 (D8 + 洼地填充 + 流量累积 + 河道路径追踪)
├── 气候模型 (纬度+地形雨+雨影+ITCZ+海岸距离+大陆度)
├── Köppen 分类 + Holdridge 判定
├── WRB 土壤推断 + 植物推荐
└── 图钉分析 (经纬度/海拔/坡度坡向/动物/矿产/城镇潜力)
```

## 核心功能

- **4 种雕刻笔刷**：隆起、削低、平滑、冰川侵蚀（三次方强度曲线）
- **4 个真实地形预设**：类地大陆、俯冲岛弧、裂谷、盘古超大陆
- **卫星图级着色**：11 级海洋深度 + 7 级陆地渐变 + 坡度纹理 + 大气透视
- **纬度驱动气候带**：赤道绿→副热带沙漠→温带绿→极地冰盖，雪线随纬度下降
- **实时等高线**：2D 模式下 fragment shader 等高线 + 高程图例 + 比例尺
- **水文模拟**：D8 流向 + 洼地填充 + 河道路径追踪 + 湖泊洪水填充
- **河流动画**：CatmullRom 平滑曲线 + 流动着色器
- **湖泊动画**：波纹着色器 + 填充多边形
- **Köppen-Geiger 气候分类**：完整 A/B/C/D/E 五类判定
- **Holdridge 生命带**：BT/PER 计算 + 生物群落推断
- **WRB 土壤推断**：16 条规则基于气候+地形+海拔
- **原生植被 + 农作物推荐**：7 种气候的植物数据库
- **代表性野生动物**：按气候带匹配（含体型/食性/适应性）
- **矿产分布推断**：6 条成矿规则基于地形+海拔+气候
- **城镇发展潜力**：7 维度评分（水/农/贸/矿/地/气候/灾害）
- **图钉系统**：放置→模拟→悬停查看 11 项地方诊断
- **撤销/重做**：20 步环形缓冲区 (Ctrl+Z/Y)
- **分享 URL**：LZ-String 压缩高度图到 URL hash
- **暗色 UI**：毛玻璃面板 + 琥珀强调色 + DM Mono 等宽字体
