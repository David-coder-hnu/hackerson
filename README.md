# TerraDiagnosis / 地脉·镜

> **推沙成世界，一镜诊山河。** —— 地理世界观 OC 构建工具

想象一个世界。用手指推起山脉，挖开海峡，让季风吹过大陆，看河流在大地上蜿蜒。放置图钉标记你的城邦，画下王国的边界。这一切都是科学可信的——你的世界有自己的气候带、土壤类型、原生植被、野生动物，甚至矿产分布和城镇发展潜力。

**TerraDiagnosis** 正是为此而生。

---

## 它是怎么工作的

### 1. 创作 —— 雕刻你的星球

打开浏览器，你面对的是整个星球的画布（360° 经度 × 180° 纬度）。用 5 种雕刻笔刷塑造大地：

| 笔刷 | 效果 |
|------|------|
| **隆起** | 抬升地形，建造山脉 |
| **削低** | 降低地形，开凿海峡 |
| **平滑** | 抚平棱角，柔化过渡 |
| **冰川** | 不规则侵蚀，雕出北欧峡湾 |
| **图钉** | 标记位置，锁定模拟后可查看 11 项地方诊断 |

四种真实地质预设可供选择：**类地大陆**（西部科迪勒拉山系 + 中央平原 + 东部古老山地）、**俯冲岛弧**（火山弧 + 深海沟）、**裂谷**（板块撕裂 + 断层崖 + 火山峰）、**盘古超大陆**（内陆沙漠 + 海岸湿润 + 中央山脉）。

### 2. 诊断 —— 让世界活过来

点击 **锁定地形**，Web Worker 启动完整的地理模拟引擎：

```
水文模拟 (D8 流向 + 洼地填充 + 河道路径追踪)
    │
    ▼
气候模拟 (纬度驱动温度 + 地形雨 + 雨影 + 距海距离 + 大陆度)
    │
    ▼
Köppen-Geiger 气候分类 (A/B/C/D/E 五类完整判定)
    │
    ▼
Holdridge 生命带 (BT 生物温度 + PER 蒸散率)
    │
    ▼
WRB 土壤推断 (16 条气候-地形-海拔规则)
    │
    ▼
原生植被 + 野生动物 + 矿产分布 + 城镇发展潜力
```

几秒后，你的星球活了过来——地形上覆盖着气候带色彩（深绿雨林到金黄沙漠），河流沿着 D8 流向从高山奔向海洋，湖泊在洼地中浮现。切换到 **2D 等高线模式**，你可以看到国家地理风格的分层设色图，完整的高程图例和比例尺。

### 3. 构建 —— 书写你的世界观

这是 TerraDiagnosis 作为 **文娱创作工具** 的核心功能：

- **分析图钉**：在编辑模式下放置图钉，锁定地形后悬停即可查看——经纬度、海拔、坡度坡向、距海岸距离、所处气压带、Köppen 气候类型、Holdridge 生物群落、年均温与年降水、土壤类型、原生植物、适宜农作物。
- **自定义图钉**：在观察模式下放置，输入文字内容。悬停显示。标记你的城市、秘境、战场。
- **自定义选区**：点击多个顶点圈定区域，命名后金色边界线 + 区域名称标签显示在地图上。画出你的王国、行省、生态区。

## 全功能一览

### 雕刻工具

| 按钮 | 功能 | 细节 |
|------|------|------|
| 视角 | 相机模式 | 左键旋转、右键平移、滚轮缩放、方向键平移、Q/E 旋转 |
| 隆起 | 高斯抬升 | 三次方强度曲线，低端极细腻 |
| 削低 | 高斯降低 | 同上 |
| 平滑 | 邻域平均 | 8× 强度加成，效果明显 |
| 冰川 | 不规则侵蚀 | 哈希平滑噪声，笔刷尺度变化，造峡湾/冰川谷 |
| 2D/3D | 视图切换 | 卫星图 ↔ 等高线分层设色图（滚轮拉远自动切） |
| 重置 | 清空 | 清除地形、标记、图钉、选区，回到浅海画布 |

### 模拟输出

| 模块 | 内容 |
|------|------|
| 地形分类 | 山脉/高原/平原/盆地百分比（百分位动态阈值） |
| 水文 | D8 河流追踪路径格点数 + 湖泊洪水填充区域格点数 + 流域覆盖率 |
| 土壤 | WRB 国际分类，含中文名、置信度、成土条件说明 |
| 植被 | 按气候带+生境匹配的原生物种，含用途（食物/建材/药材/纤维） |
| 总结语 | 动态生成的「你创造了一个…的世界」自然语言描述 |

### 渲染特性

- **3D 卫星图**：6 级地形分层设色（深草绿→亮黄绿→金黄→赤褐→红棕→纯白）+ 3 级海洋深度（浅海蓝→深海蓝→深渊蓝）+ 5 级植被密度（荒漠→草原→农田→森林→雨林）
- **等高线图**：计曲线（5 倍间隔，粗黑线）+ 首曲线（1 倍间隔，中灰线）+ 高程数字标签（每 500m）+ 底部高程图例 + 比例尺
- **纬度驱动**：雪线随纬度下降（赤道 5000m → 极地海平面），极地冰盖 + 亚极地苔原
- **水体动画**：河流流动着色器 + 湖泊波纹着色器
- **大气效果**：视距雾化 + 高海拔蓝色散射

## 技术架构

```
浏览器主线程
├── React UI (工具条/参数面板/状态栏/档案面板/图钉卡片)
├── React Three Fiber 3D 场景
│   ├── TerrainMaterial (自定义 GLSL, 512×512 DataTexture)
│   ├── WaterMaterial (动画着色器, uTime 驱动)
│   ├── 河流 CatmullRom 曲线 + 湖泊 ShapeGeometry
│   ├── 图钉锥形模型 + 选区边界线 + 区域名 Text 标签
│   └── OrbitControls (动态 maxPolarAngle, 俯瞰自动切换)
└── Zustand Store { heightmap, brush, markers, simProgress, archive, ... }
            │
            │ postMessage (Float32Array transfer, 零拷贝)
            ▼
Web Worker 模拟引擎
├── 水文: fillDepressions → D8 流向 → 流量累积 → 河道路径追踪 → 湖泊洪水填充
├── 气候: BFS 距海距离 → 纬温 + 地形雨(多步回看) + 雨影 + ITCZ + 大陆度
├── 分类: Köppen-Geiger + Holdridge 生命带 + WRB 土壤 + 图钉分析
└── 知识库: 14 个地理知识模块 (JSON + Markdown)
```

## 知识库

项目包含一个由团队成员独立构建的 **14 模块地理知识库**（`知识库/` 目录），涵盖：

| 模块 | 内容 |
|------|------|
| 全球气候形成综合框架 | 五层驱动模型 (纬度→大气环流→海陆→洋流→地形) |
| 地形-气候转化规则引擎 | 海拔/坡度/坡向/距水距离 → 温度/降水/大陆度 |
| 柯本气候类型判定矩阵 | 完整 A/B/C/D/E 分类 + 成因链 |
| 生态-农业-土壤综合诊断 | Holdridge + FAO AEZ + WRB 三层推理 |
| 气候-土壤对应规则库 | 每种气候 → 典型/山地/河谷土壤变体 (JSON) |
| 气候-原产植物库 | 按气候+生境的植物种类 + 用途 (JSON) |
| 气候-野生动物属库 | 按气候带的动物属 + 体型/食性/适应性 (JSON) |
| 矿产分布规则库 | 地形+岩石+构造+气候 → 矿种推断 (JSON) |
| 城镇发展潜力评估引擎 | 7 维度加权评分 + 适宜产业推荐 (JSON) |
| 基础地理参数推断引擎 | 6 项用户输入 → 自动推断月温/降水/母质/排水 |
| 水文推断引擎 | Strahler 河流等级/湖泊类型/洪水风险 |
| 测试管线 | 斯德哥尔摩/38°N106°E/59°N18°E 三地验证 |

## 快速开始

```bash
git clone https://github.com/David-coder-hnu/hackerson.git
cd hackerson
npm install
npm run dev
```

浏览器打开 `http://localhost:5173`。选择预设 → 推拉地形 → 点击「锁定地形」→ 探索你的世界。
